## 1. 快速排序
>思想：分治

步骤：
1. 确定分界点x（均可）
2. **调整区间** （保证左<=x，右>=x）
3. 递归处理左右区间

```cpp
void quick_sort(int q[], int l, int r)
{
    if (l >= r) return;

    int i = l - 1, j = r + 1, x = q[l + r >> 1]; // 每次执行时指针会分别向中间移一次，因此初始化需要在数组左右区间往外一格
    while (i < j) // 指针相遇时排序完毕
    {
        do i ++ ; while (q[i] < x);
        do j -- ; while (q[j] > x);
        if (i < j) swap(q[i], q[j]); // 两指针均满足左右与x的大小关系后，执行交换
    }
    quick_sort(q, l, j), quick_sort(q, j + 1, r); // 分界一般不用i和j，而直接用j和j+1（因为结束循环的条件是相遇或i超过j）
}
```

## 2. 归并排序
步骤：
1. 确定分界点mid（只能取中点）
2. 递归处理左右区间
3. **归并**：合二为一

```cpp
void merge_sort(int q[], int l, int r)
{
    if (l >= r) return;

    int mid = l + r >> 1;
    merge_sort(q, l, mid);
    merge_sort(q, mid + 1, r);

    int k = 0, i = l, j = mid + 1;
    while (i <= mid && j <= r) // 分别遍历左右数组
        if (q[i] <= q[j]) tmp[k ++ ] = q[i ++ ];
        else tmp[k ++ ] = q[j ++ ]; // 比较左右数组当前大小，满足的移入结果数组

    while (i <= mid) tmp[k ++ ] = q[i ++ ]; 
    while (j <= r) tmp[k ++ ] = q[j ++ ]; // 循环完左右的数组，直到全部结束

    for (i = l, j = 0; i <= r; i ++, j ++ ) q[i] = tmp[j]; // 将结果数组存入原数组
}
```
