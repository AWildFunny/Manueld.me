本质：寻找边界，使得左右区间一个满足、一个不满足条件
## 1. 整数二分

两种情况对边界进行处理

```cpp
bool check(int x) {/* ... */} // 检查x是否满足某种性质

// 满足条件时，mid在左，即区间[l, r]被划分成[l, mid]和[mid + 1, r]时使用：
int bsearch_1(int l, int r)
{
    while (l < r)
    {
        int mid = l + r >> 1; // >>1即二进制数右移一位，相当于除以2向下取整
        if (check(mid)) r = mid;    // check()判断mid是否满足性质
        else l = mid + 1;
    }
    return l;
}
// 满足条件时，mid在右，即区间[l, r]被划分成[l, mid - 1]和[mid, r]时使用：
int bsearch_2(int l, int r)
{
    while (l < r)
    {
        int mid = l + r + 1 >> 1; // 需要补上+1，否则会导致死循环
        if (check(mid)) l = mid;
        else r = mid - 1;
    }
    return l;
}
```

## 2. 浮点数二分

不需要判断边界，更方便

```cpp
bool check(double x) {/* ... */} // 检查x是否满足某种性质

double bsearch_3(double l, double r)
{
    const double eps = 1e-6;   // eps 表示精度，取决于题目对精度的要求
    while (r - l > eps)
    {
        double mid = (l + r) / 2;
        if (check(mid)) r = mid;
        else l = mid;
    }
    return l;
}
```
