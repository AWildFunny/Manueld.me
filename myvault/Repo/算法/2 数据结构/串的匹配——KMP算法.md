
对于待匹配的一个字符串s：
- 前缀：除尾部字符外，串的所有**头部字串**
- 后缀：除头部字符外，串的所有**尾部字串**

对于串s的**每个字符**：
- 部分匹配值（**PM**,Partial Match）= 该字符及其前所有字符组成的串中，前缀&后缀的**最长相等长度**

对于串s，计算得到PM表（每个字符对应的PM值）后：
- next数组：next[ j ] 代表**模式串的第j个字符失配时，跳到next[ j ]位置继续比较**
- next[ j ] = PM[ j-1 ] + 1，即PM表右移一位、整体+1得到next数组
- next数组**前两位固定为0、1**

>上述均假设串编号始于1，若始于0则均需要-1

计算机中实现next数组计算方法：
1. 初始化前两位为0、1，然后循环主串的每一位 j：
	1. 对于模式串的每一位k：
		1. 判断其是否与主串的前一位 j-1 相等
		2. 相等 -> next[ j+1 ] = next[ j ] + 1跳出循环
		3. 不相等 -> j = next[ j ]继续循环

改进：nextval数组
1. 第一位默认为0
2. 检查next数组的每位，找出其对应的模式串中应该匹配的字符S（即next[j]对应的字符）：
	1. 若S与该位原本的字符S不相同：则保留nextval[j] = next[j]的值，因为此时确实应该使用该值进行回溯
	2. 若相同：则nextval[j] = nextval[next[j]]
