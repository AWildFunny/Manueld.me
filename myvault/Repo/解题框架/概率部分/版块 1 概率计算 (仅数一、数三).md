# 版块 1 概率计算(仅数一、数三)

## A. 实用解题工具

### 第 1 组: 事件基本公式

1. 减法: $P\left( {A - B}\right)  = P\left( A\right)  - P\left( {AB}\right)$ ;

2. 加法: $\left\{  \begin{array}{ll} P\left( {A + B}\right)  = P\left( A\right)  + P\left( B\right)  - P\left( {AB}\right) \\  P\left( {A + B + C}\right)  = P\left( A\right)  + P\left( B\right)  + P\left( C\right)  - P\left( {AB}\right)  - P\left( {BC}\right)  - P\left( {AC}\right)  + P\left( {ABC}\right)  \end{array}\right.$

3. 条件概率: $P\left( {B \mid  A}\right)  = \frac{P\left( {AB}\right) }{P\left( A\right) }$ ,表示 $A$ 发生的条件下, $B$ 发生的概率

4. 全概率: $P\left( B\right)  = \mathop{\sum }\limits_{{i = 1}}^{n}P\left( {B \mid  {A}_{i}}\right) P\left( {A}_{i}\right) ,{A}_{i}{A}_{j} = \varnothing ,i \neq  j,\mathop{\bigcup }\limits_{{i = 1}}^{n}{A}_{i} = \Omega$ .

5. 贝叶斯基本公式: $P\left( {{A}_{j} \mid  B}\right)  = \frac{P\left( {B \mid  {A}_{j}}\right) P\left( {A}_{j}\right) }{\mathop{\sum }\limits_{{i = 1}}^{n}P\left( {B \mid  {A}_{i}}\right) P\left( {A}_{i}\right) },j = 1,2,\cdots ,n$

6. 乘法: $\left\{  \begin{array}{lll} \text{ 情况 }1 : P\left( {AB}\right)  = P\left( A\right) P\left( {B \mid  A}\right) \\  \text{ 情况 }2 : P\left( {A\bar{B}}\right)  = P\left( A\right)  - P\left( {AB}\right) \\  \text{ 情况 }3 : P\left( {AB}\right)  = P\left( A\right) P\left( B\right) \left( {A,B\text{ 独立 }}\right)  \end{array}\right.$

7. 德摩根律: $\overline{A \cup  B} = \bar{A} \cap  \bar{B},\overline{A \cap  B} = \bar{A} \cup  \bar{B}$

8. 相互独立: $P\left( {AB}\right)  = P\left( A\right) P\left( B\right)$

1. 分布函数 $\left\{  \begin{array}{ll} \text{ 性质: }\left\{  \begin{array}{lll} 0 \leq  F\left( x\right)  \leq  1;F\left( {-\infty }\right)  = 0,F\left( {+\infty }\right)  = 1 \\  F\left( x\right) \text{ 单调不减 } \\  \text{ 右连续 }F\left( {x + 0}\right)  = F\left( x\right)  \end{array}\right. \\  \text{ 公式 : }P\left\{  {{x}_{1} < X \leq  {x}_{2}}\right\}   = F\left( {x}_{2}\right)  - F\left( {x}_{1}\right)  \end{array}\right.$

2. 概率密度 $\left\{  \begin{array}{ll} \text{ 性质 }\left\{  \begin{array}{ll} f\left( x\right)  \geq  0 \\  {\int }_{-\infty }^{+\infty }f\left( x\right) {dx} = 1 \end{array}\right. \\  \text{ 公式 } : P\{ X \leq  x\}  = F\left( x\right)  = {\int }_{-\infty }^{x}f\left( t\right) {dt} \end{array}\right.$

4. 指数分布 $\left\{  \begin{array}{lllll} \text{ 概率密度函数 }f\left( x\right)  = \left\{  \begin{matrix} \lambda {e}^{-{\lambda x}}, & x \geq  0, \\  0, & x < 0 \end{matrix}\right. \left( {\lambda  > 0}\right) \text{,记作 }X \sim  e\left( \lambda \right) \\  \text{ 分布函数 }F\left( x\right)  = {\int }_{-\infty }^{x}f\left( t\right) {dt} = \left\{  \begin{matrix} 1 - {e}^{-{\lambda x}}, & x > 0, \\  0, & x \leq  0 \end{matrix}\right. \left( {\lambda  > 0}\right) \\  \text{ 5.正态分布 } \\  \text{ 5.正态分布 } \\  \text{ 标准正态的分布函数 }\Phi \left( x\right)  = {\int }_{-\infty }^{x}\frac{{e}^{-\frac{{\left( x - \mu \right) }^{2}}{2{\sigma }^{2}}}}{}, - \infty  < x <  + \infty ,\text{ 记作 }X \sim  N\left( {\mu ,{\sigma }^{2}}\right)  \end{array}\right.$

### 第 3 组: 一维离散的分布

1. 二维离散 $\left\{  \begin{array}{llll} \text{ 1.联合概率分布: }P\left\{  {X = {x}_{i},Y = {y}_{j}}\right\}   = {p}_{ij};i,j = 1,2,\cdots \\  \text{ 2.边缘分布: }{p}_{i \cdot  } = \mathop{\sum }\limits_{{j = 1}}^{\infty }{p}_{ij},i = 1,2,\cdots ,{p}_{\cdot j} = \mathop{\sum }\limits_{i}^{\infty }{p}_{ij},j = 1,2,\cdots \\  \text{ 3.条件分布律 : }P\left\{  {X = {x}_{i} \mid  Y = {y}_{j}}\right\}   = \frac{{p}_{ij}}{{p}_{\cdot j}},P\left\{  {Y = {y}_{j} \mid  X = {x}_{i}}\right\}   = \frac{{p}_{ij}}{{p}_{i \cdot  }} \\  \text{ 4. 独立性 : 若对一切 }i,j,\text{ 有 }{p}_{ij} = {p}_{i \cdot  } \cdot  {p}_{\cdot j}\text{,则称 }X\text{ 与 }Y\text{相互独立 } \end{array}\right.$

2. 几类一维离散分布

（1）泊松分布 $p\left( \lambda \right)  : P\left( {X = k}\right)  = \frac{{\lambda }^{k}}{k!}{e}^{-\lambda },\lambda  > 0,k = 0,1,2\cdots$

（2） 0-1 分布: $P\left( {X = k}\right)  = {p}^{k}{\left( 1 - p\right) }^{1 - k},k = 0,1$

（3）二项分布 $B\left( {n,p}\right)  : P\left( {X = k}\right)  = {C}_{n}^{k}{p}^{k}{\left( 1 - p\right) }^{n - k},k = 0,1,\cdots ,n$

（4）几何分布 $G\left( p\right)  : P\left( {X = k}\right)  = {\left( 1 - p\right) }^{k - 1}p,0 < p < 1,k = 1,2,\cdots$ . (第 $k$ 次成功的概率)

$\left\{  \begin{array}{l} \text{ 1. }F\left( {x,y}\right) \text{ 的性质: }\left\{  \begin{array}{llll} 0 \leq  F\left( {x,y}\right)  \leq  1,F\left( {-\infty ,y}\right)  = F\left( {x, - \infty }\right)  = 0,F\left( {+\infty , + \infty }\right)  = 1 \\  \text{ 分别关于 }x\text{ 和 }y\text{ 单调不减 } \\  F\left( {x + 0,y}\right)  = F\left( {x,y}\right) ,F\left( {x,y + 0}\right)  = F\left( {x,y}\right) \\  P\{ \left( {X,Y}\right)  \in  D\}  = F\left( {{x}_{2},{y}_{2}}\right)  - F\left( {{x}_{2},{y}_{1}}\right)  - F\left( {{x}_{1},{y}_{2}}\right)  + F\left( {{x}_{1},{y}_{1}}\right)  \end{array}\right.  \end{array}\right.$

2. $f\left( {x,y}\right)$ 的性质: $\left\{  \begin{array}{ll} f\left( {x,y}\right)  \geq  0 \\  {\int }_{-\infty }^{+\infty }{\int }_{-\infty }^{+\infty }f\left( {x,y}\right) {dxdy} = 1 \end{array}\right.$

3. 边缘概率密度: ${f}_{X}\left( x\right)  = {\int }_{-\infty }^{+\infty }f\left( {x,y}\right) {dy};{f}_{Y}\left( y\right)  = {\int }_{-\infty }^{+\infty }f\left( {x,y}\right) {dx}$

4. 条件概率密度: ${f}_{X \mid  Y}\left( {x \mid  y}\right)  = \frac{f\left( {x,y}\right) }{{f}_{Y}\left( y\right) };{f}_{Y \mid  X}\left( {y \mid  x}\right)  = \frac{f\left( {x,y}\right) }{{f}_{X}\left( x\right) }$

5. 独立性验证: $X$ 与 $Y$ 相互独立 $\Leftrightarrow  \left\{  \begin{array}{lll} f\left( {x,y}\right)  = {f}_{X}\left( x\right)  \cdot  {f}_{Y}\left( y\right) \\  {f}_{X \mid  Y}\left( {x \mid  y}\right)  = {f}_{X}\left( x\right) \\  {f}_{Y \mid  X}\left( {y \mid  x}\right)  = {f}_{Y}\left( y\right)  \end{array}\right.$

6. 二维均匀分布: $f\left( {x,y}\right)  = \left\{  \begin{array}{ll} \frac{1}{S\left( D\right) },\left( {x,y}\right)  \in  D \\  0,\text{ 其他 } \end{array}\right.$

7. 二维正态: $\begin{cases}  & \text{ 表示符号 : }N\left( {{\mu }_{1},{\mu }_{2},{\sigma }_{1}^{2},{\sigma }_{2}^{2},\rho }\right) \\   & f\left( {x,y}\right)  = \\   & \frac{1}{{2\pi }{\sigma }_{1}{\sigma }_{2}\sqrt{1 - {\rho }^{2}}}\exp \left\{  {\frac{-1}{2\left( {1 - {\rho }^{2}}\right) }\left\lbrack  {\frac{{\left( x - {\mu }_{1}\right) }^{2}}{{\sigma }_{1}^{2}} - {2\rho }\frac{\left( {x - {\mu }_{1}}\right) \left( {y - {\mu }_{2}}\right) }{{\sigma }_{1}{\sigma }_{2}} + \frac{{\left( y - {\mu }_{2}\right) }^{2}}{{\sigma }_{2}^{2}}}\right\rbrack  }\right\}  \\   & X \sim  N\left( {{\mu }_{1},{\sigma }_{1}^{2}}\right) ,Y \sim  N\left( {{\mu }_{2},{\sigma }_{2}^{2}}\right) \\   & X\text{ 与 }Y\text{ 相互独立 } \\   & X\text{,}Y\text{ 线性组合依然服从正态 } \\   & \left( {{aX} + {bY},{cX} + {dY}}\right) \text{ 服从二维正态 }\left( {{aX} + {bY}\text{ 与 }{cX} + {dY}\text{ 不成比例 }}\right) \\   & \left( {{aX} + {bY},{cX} + {dY}}\right) \text{ 服从二维正态 }\left( {{aX} + {bY}\text{ 与 }{cX} + {dY}\text{ 不成比例 }}\right)  \end{cases}$

### 第 7 组: 独立性验证

独立性验证  $\left\{  \begin{array}{lllll} \text{ ① }P\left( {AB}\right)  = P\left( A\right) P\left( B\right) \\  \text{ ② }F\left( {x,y}\right)  = {F}_{X}\left( x\right) {F}_{Y}\left( y\right) \\  \text{ ③ }f\left( {x,y}\right)  = {f}_{X}\left( x\right) {f}_{Y}\left( y\right) \\  \text{ ④ }{p}_{ij} = {p}_{i \cdot  }{p}_{\cdot j} \\  \text{ ⑤ }\rho  = 0\text{,仅限于二维正态 } \end{array}\right.$

例题 1.已知随机变量 $X$ 服从标准正态分布 $N\left( {0,1}\right)$ ,则 $X$ 与 ${X}^{2}$ （ $\;$ ）.

(A) 不相关且相互独立 (B) 不相关且相互不独立

(C) 相关且相互独立 (D) 相关且相互不独立

【解】选 B. 由于 $X \sim  N\left( {0,1}\right)$ ,则 $E\left( X\right)  = 0,E\left( {X}^{2}\right)  = D\left( X\right)  = 1,E\left( {X}^{3}\right)  = 0$ ,

于是 $\operatorname{Cov}\left( {X,{X}^{2}}\right)  = E\left( {X{X}^{2}}\right)  - E\left( X\right) E\left( {X}^{2}\right)  = E\left( {X}^{3}\right)  - E\left( X\right) E\left( {X}^{2}\right)  = 0$ ,故 $X$ 与 ${X}^{2}$ 不相关.

下面证明 $X$ 与 ${X}^{2}$ 相互不独立,这只需证明存在实数 $a,b$ ,使得事件 $\{ X \leq  a\}$ 与 $\left\{  {{X}^{2} \leq  b}\right\}$ 不独立即

可. 取 $a = \frac{1}{2},b = \frac{1}{4}$ ,则 $P\left\{  {X \leq  \frac{1}{2},{X}^{2} \leq  \frac{1}{4}}\right\}   = P\left\{  {{X}^{2} \leq  \frac{1}{4}} \right\}   = P\left\{  {-\frac{1}{2} \leq  X \leq  \frac{1}{2}}\right\}   = {2\Phi }\left( \frac{1}{2}\right)  - 1$ ,

又 $P\left\{  {X \leq  \frac{1}{2}}\right\}   = \Phi \left( \frac{1}{2}\right)$ ,因此, $P\left\{  {X \leq  \frac{1}{2},{X}^{2} \leq  \frac{1}{4}}\right\}   \neq  P\left\{  {X \leq  \frac{1}{2}}\right\}  P\left\{  {{X}^{2} \leq  \frac{1}{4}} \right\}$ ,

所以 $X$ 与 ${X}^{2}$ 相互不独立.

## B. 核心问题及解题方法总结

 **1. 求解泊松分布

泊松分布  $\begin{cases} P\{ X = k\}  = \frac {{\lambda }^{k}{e}^{-\lambda }} {k!}, & k = 0,1,2,\cdots \text{ 记作 }X \sim  P\left( \lambda \right) \left( {\lambda  > 0}\right) \\  \text{ 泊松定理 } &  : \mathop{\lim }\limits_ {{n \rightarrow  \infty }} {C}_{n}^{k}{p}_{n}^{k}{\left( 1 - {p}_{n}\right) }^{n - k} = \frac {{\lambda }^{k}{e}^{-\lambda }} {k!}\left( {\mathop{\lim }\limits_ {{n \rightarrow  \infty }} n{p}_{n} = \lambda  > 0}\right) \\   & \text{ 含义: 当 }n\text{ 很大,}p\text{ 很小时,二项分布可用泊松分布逼近 } \end{cases}$

例题 1.设随机变量 $X$ 服从参数为 1 的泊松分布,则 $P\left\{  {X = E{X}^{2}}\right\}   = \;$ ___

【解】对于 $X \sim  P\left( \lambda \right)$ 有 $P\{ X = k\}  = \frac {{\lambda }^{k}} {k!}{e}^{-\lambda },k = 0,1,2,\ldots ,E\left( X\right)  = D\left( X\right)  = \lambda$ ,先将 $\lambda  = 1$ 直接代入即可。

$$
E\left( {X}^{2}\right)  = D\left( X\right)  + {E}^{2}\left( X\right)  = \lambda  + {\lambda }^{2} = 2
$$

$$
P\left\{  {X = E{X}^{2}}\right\}   = P\{ X = 2\}  = \frac {{1}^{2}} {2!}{e}^{-1} = \frac{1}{2e}\text{,填 }\frac{1}{2e}
$$

 **2. 求解其他一维离散

一维离散 $\left\{  \begin{array}{ll} \text{ 1. 几何分布 }\left\{  \begin{array}{ll} P\{ X = k\}  = p{\left( 1 - p\right) }^{k - 1},k = 1,2,\cdots ,\text{ 记作 }X \sim  G\left( p\right) \left( {0 < p < 1}\right) \\  \text{ 含义: }X\text{ 为事件 }A\text{ 首次发生时的试验次数 } \end{array}\right. \\  \text{ 2.二项分布 }\left\{  \begin{array}{ll} P\left( {X = k}\right)  = {C}_{n}^{k}{p}^{k}{\left( 1 - p\right) }^{n - k},\text{ 记作 }X \sim  B\left( {n,p}\right) \left( {0 < p < 1}\right) \\  \text{ 含义: }n\text{ 次独立重复试验中事件 }A\text{ 发生 }n\text{次的概率 } \end{array}\right.  \end{array}\right.$

函数的分布: $P\left( {X = {x}_{1}}\right)  = {p}_{i},Y = g\left( X\right)$ ,则 $P\left( {Y = {y}_{j}}\right)  = \mathop{\sum }\limits_ {{g\left( {x}_{i}\right)  = {y}_{i}} }P\left( {X = {x}_{i}}\right)$

 **3. 求解指数分布

指数分布 $\left\{  \begin{array}{lll} \text{ 概率密度函数 }f\left( x\right)  = \left\{  {\begin{matrix} \lambda {e}^{-{\lambda x}}, & x \geq  0, \\  0,\Delta & x < 0 \end{matrix}\left( {\lambda  > 0}\right) \text{,记作 }X \sim  e(}\right. \\  \text{ 分布函数 }F\left( x\right)  = {\int }_{-\infty }^{x}f\left( x\right) {dx} = \left\{  {\begin{matrix} 1 - {e}^{-{\lambda x}}, & x > 0, \\  0, & x \leq  0 \end{matrix}\left( {\lambda  > 0}\right) }\right. \\  \text{ 数字特征: }{EX} = \frac{1}{\lambda },{DX} = \frac{1} {{\lambda }^{2}} \end{array}\right.$

例题 1.设随机变量 $X$ 服从参数为 $\lambda$ 的指数分布,则 $P\{ X > \sqrt{DX}\}  = \underline{\;}$ .

【解】指数分布的概率密度为: $f\left( x\right)  = \left\{  {\begin{matrix} \lambda {e}^{-{\lambda x}},x > 0 \\  0,x \leq  0 \end{matrix},\left( {\lambda  > 0}\right) }\right.$ ,

分布函数为: $F\left( x\right)  = \left\{  \begin{matrix} 1 - {e}^{-{\lambda x}},x \geq  0 \\  0,x < 0 \end{matrix}\right.$ ,方差为 $D\left( X\right)  = \frac{1} {{\lambda }^{2}} $

$P\{ X > \sqrt{DX}\}  = P\left\{  {X > \frac{1}{\lambda }}\right\}   = {\int }_{\frac{1}{\lambda }}^{+\infty }\lambda {e}^{-{\lambda x}}{dx} = {e}^{-1},$

或 $P\{ X > \sqrt{DX}\}  = P\left\{  {X > \frac{1}{\lambda }}\right\}   = 1 - F\left( \frac{1}{\lambda }\right)  = 1 - \left( {1 - {e}^{-\lambda  \cdot  \frac{1}{\lambda }}}\right)  = {e}^{-1}$ ,故填 ${e}^{-1}$ 。

 **4. 求解正态分布

1. 标准正态: $\left\{  \begin{array}{ll} X \sim  N\left( {0,1}\right) ,\text{ 分布函数 }\Phi \left( x\right)  = {\int }_{-\infty }^{x}\frac{1}{\sqrt{2\pi }}{e}^{-\frac {{t}^{2}} {2}}{dt} \\  \Phi \left( a\right)  + \Phi \left( {-a}\right)  = 1 \Rightarrow  \Phi \left( a\right)  - \Phi \left( {-a}\right)  = {2\Phi }\left( a\right)  - 1 \end{array}\right. \varnothing$

2. 密度: $\varphi \left( x\right)  = \frac{1}{\sqrt{2\pi }\sigma }{e}^{-\frac {{\left( x - \mu \right) }^{2}} {2{\sigma }^{2}}}, - \infty  < x <  + \infty$ ,记作 $X \sim  N\left( {\mu ,{\sigma }^{2}}\right)$

 **5.分位数**: $\left\{  \begin{array}{ll} X \sim  N\left( {0,1}\right) \text{,实数 }{u}_{\alpha }\text{ 满足 }P\left( {X \geq  {u}_{\alpha }}\right)  = \alpha \text{,则称 }{u}_{\alpha }\text{ 为 }\alpha \text{ 的上侧分位数 } \\  X \sim  N\left( {0,1}\right) \text{,实数 }{u}_{\frac{\alpha }{2}}\text{ 满足 }P\left( {\left| X\right|  > {u}_{\frac{\alpha }{2}}}\right)  = \alpha \text{,则称 }{u}_{\frac{\alpha }{2}}\text{ 为 }\alpha \text{的双侧分位数 } \end{array}\right.$

例题 1. 设随机变量 $X$ 服从正态分布 $N\left( {0,1}\right)$ ,对给定的 $\alpha \left( {0 < \alpha  < 1}\right)$ ,数 ${u}_{\alpha }$ 满足 $P\left\{  {X > {u}_{\alpha }}\right\}   = \alpha$ ,若

$P\{ \left| X\right|  < x\}  = \alpha$ ,则 $x$ 等于 ( )
