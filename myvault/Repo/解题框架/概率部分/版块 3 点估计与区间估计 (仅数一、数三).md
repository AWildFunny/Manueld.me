# 版块 3 点估计与区间估计(仅数一、数三)

### 第 1 组: 无偏估计/有效性/一致性 (仅数学一)

无偏性: $E\left( \widehat{\theta }\right)  = \theta$

有效性: ${\widehat{\theta }}_{1}$ 与 ${\widehat{\theta }}_{2}$ 都是 $\theta$ 的无偏估计量,

若 $D\left( {\widehat{\theta }}_{1}\right)  < D\left( {\widehat{\theta }}_{2}\right)$ ,则称 ${\widehat{\theta }}_{1}$ 较 ${\widehat{\theta }}_{2}$ 有效

一致性: 当 $n \rightarrow  \infty$ 时, $\widehat{\theta }$ 依概率收敛于 $\theta$

则称 $\widehat{\theta }$ 为 $\theta$ 的一致估计量

### 第 2 组: 估计的性质

（1）若 $\widehat{\theta }$ 为 $\theta$ 的矩估计, $g\left( x\right)$ 为连续函数,则 $g\left( \widehat{\theta }\right)$ 为 $g\left( \theta \right)$ 的矩估计.

（2） $\widehat{\theta }$ 为 $\theta$ 的极大似然估计， $g\left( x\right)$ 为单调函数,则 $g\left( \widehat{\theta }\right)$ 为 $g\left( \theta \right)$ 的极大似然估计

（3）（仅数一） $E\left( \bar{X}\right)  = E\left( X\right) ,E\left( {S}^{2}\right)  = D\left( X\right)$ ,即 $\bar{X},{S}^{2}$ 分别为总体 $E\left( X\right) ,D\left( X\right)$ 的无偏估计量.

（4）（仅数一）由大数定律易知 $\bar{X},{S}^{2}$ 也分别是 $E\left( X\right) ,D\left( X\right)$ 的一致估量.

## B. 核心问题及解题方法总结

 **1. 计算点估计
$\text{点估计}\left\{  \begin{array}{llll} \text{ 矩估计 : }E\left( {X}^{n}\right)  = {A}_{n} = \frac{1}{n}\mathop{\sum }\limits_ {{i = 1}} ^{n}{X}_{i}^{n} \Rightarrow  \theta  = \bar{\theta }\left( {{X}_{1},{X}_{2},\ldots ,{X}_{n}} \right) \\  \text{ 极大似然估计 : }\left\{  \begin{array}{ll} \text{ 似然函数: 连续型 : }L\left( \theta \right)  = \mathop{\prod }\limits_ {{i = 1}} ^{n}f\left( {{x}_{i};\theta }\right) \text{; 离散型 : }L\left( \theta \right)  = \mathop{\prod }\limits_{{i = 1}} ^{n}P\left( {{x}_{i};\theta }\right) \\  \text{ 求最值 : }\left\{  \begin{array}{lll} \text{ 有驻点: }\ln L\left( \theta \right)  = \mathop{\sum }\limits_{{i = 1}} ^{n}P\left( {{x}_{i};\theta }\right) ,\frac{d\ln L\left( \theta \right) }{d\theta } = 0 \Rightarrow  \widehat{\theta } = \widehat{\theta }\left( {{x}_{1},{x}_{2},\cdots ,{x}_{n}} \right) \\  \text{ 无驻点: 根据实际关义求出最大值点 } \\   \end{array}\right.  \end{array}\right. \\  \text{ 最大似然估计的不变性: }\widehat{\theta }\text{ 是确最大似然估计,函数 }\varphi \left( \theta \right) \text{ 具有单值反函数 } \\  \varphi  = \varphi \left( \widehat{\theta }\right) \text{ 是 }\varphi \left( \theta \right) \text{ 的最大似然估计,即 }\varphi \left( \theta \right)  = \varphi \left( \widehat{\theta }\right) \text{. } \end{array}\right.$

例题 1. 设总体 $X$ 的概率密度为 $f\left( x\right)  = \left\{  \begin{matrix}  - {e}^{-x} & ,x \geq  \mu \\  \sigma & ,x < \mu  \end{matrix}\right.$ ,其中 $\mu$ 是已知参数, $\sigma$ 是未知参数, $A$ 是常

数, ${X}_{1},{X}_{2},\cdots ,{X}_{n}$ 是来自总体 $X$ 的简单随机样本.

（1）求常数 $A$ 的值;

（2）求 ${\sigma }^{2}$ 的最大似然估计量.

【解】（1）由 ${\int }_{-\infty }^{+\infty }f\left( x\right) {dx} = 1$ 可知 ${\int }_{\mu }^{+\infty }\frac{A}{\sigma }{e}^{-\frac{{\left( x - \mu \right) }^{2}}{2{\sigma }^{2}}}{dx} = \sqrt{2}A{\int }_{0}^{+\infty }{e}^{-{\left( \frac{t}{\sqrt{2}\sigma }\right) }^{2}}d\left( \frac{t}{\sqrt{2}\sigma }\right)  = \sqrt{\frac{\pi }{2}}A = 1$ 。所以 $A = \sqrt{\frac{2}{\pi }}$ . 似然函数为 $L\left( {{X}_{1},{X}_{2},\cdots {X}_{n};{\sigma }^{2}}\right)  = \mathop{\prod }\limits_{{i = 1}}^{n}f\left( {{x}_{i},\sigma }\right)  = \left\{  \begin{matrix} \frac{{A}^{n}}{{\sigma }^{n}}{e}^{-\frac{\mathop{\sum }\limits_{{i = 1}}^{n}{\left( {X}_{i} - \mu \right) }^{2}}{2{\sigma }^{2}}},{x}_{i} \geq  \mu , \\  0,\;\text{ 其他 } \end{matrix}\right.$

取对数,得 $\ln L\left( {{X}_{1},{X}_{2},\cdots ,{X}_{n};{\sigma }^{2}}\right)  = n\ln A - \frac{n}{2}\ln \left( {\sigma }^{2}\right)  - \frac{1}{2{\sigma }^{2}}\mathop{\sum }\limits_{{i = 1}}^{n}{\left( {X}_{i} - \mu \right) }^{2}$

解方程 $\frac{d\ln L\left( {{X}_{1},{X}_{2},\cdots ,{X}_{n};{\sigma }^{2}}\right) }{d\left( {\sigma }^{2}\right) } =  - \frac{n}{2}\frac{1}{{\sigma }^{2}} + \frac{1}{2{\left( {\sigma }^{2}\right) }^{2}}\mathop{\sum }\limits_{{i = 1}}^{n}{\left( {X}_{i} - \mu \right) }^{2} = 0$ ,得未知参数 ${\sigma }^{2}$ 的最大似

解方程 $\frac{d\ln L\left( {{X}_{1},{X}_{2},\cdots ,{X}_{n};{\sigma }^{2}}\right) }{d\left( {\sigma }^{2}\right) } =  - \frac{n}{2}\frac{1}{{\sigma }^{2}} + \frac{1}{2{\left( {\sigma }^{2}\right) }^{2}}\mathop{\sum }\limits_{{i = 1}}^{n}{\left( {X}_{i} - \mu \right) }^{2} = 0$ ,得未知参数 ${\sigma }^{2}$ 的最大似

然估计量为 $\widehat{{\sigma }^{2}} = \frac{1}{n}\mathop{\sum }\limits_{{i = 1}}^{n}{\left( {X}_{i} - \mu \right) }^{2}$ .
