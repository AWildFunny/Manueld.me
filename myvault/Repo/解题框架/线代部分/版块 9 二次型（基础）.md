# 第 1 组: 实对称矩阵的性质

 实对称矩阵
1. 不同特征值对应的特征向量相互正交 
2. 一定可以对角化, 且对角阵的元素即为矩阵本身的特征值
3. 相似必合同
4. 若有$k$重特征值$\lambda$,则$\lambda$必有$k$个线性无关的特征向量
5. 非零特征值的个数等于秩 
6. <span style="background:#b1ffff">二次型的矩阵必为实对称矩阵</span>
7. 实对称矩阵必合同于实对称矩阵
# 第 2 组: 矩阵合同
 合同：
1. 惯性定理: 对于任意二次型, 不论选取怎样的合同变换使它化为标准形,其正负惯性指数与所选变换无关
2. 正 (负)惯性指数: 标准形中正 (负)平方项的个数 
3. 实对称阵$A,B$合同:
表达式:$C^{\mathrm{T}}AC=B$(其中 C 可逆)
等价于:$A,B$具有相同的正负惯性指数
4. 若$A,B$均为实对称, 且$A,B$相似, 则$A,B$必合同 
5.$A,B$合同, 必有$A,B$的秩相等
6.$A$合同于单位阵, 则$A$正定
 7.$C^{\mathrm{T}}AC=\Lambda$,求$C$
特征值法:先求特征值再求特征向量,得出$P$矩阵, 最终得正交矩阵 C
<span style="background:#b1ffff">配方法:</span>配方得出可逆变换$x= Cy$, 矩阵$C$即为所求
## B . 核心问题及解题方法总结
 **1. 正交变换将二次型化为标准形**
 正交变换
步骤: (1)求特征值;(2)求特征向量;(3)正交化单位化得到$Q$
结果:$x^{\mathrm{T}}Ax\xrightarrow{x=Qy}y^{\mathrm{T}}Q^{\mathrm{T}}AQy=y^{\mathrm{T}}\Lambda y=\lambda_1y_1^2+\lambda_2y_2^2+\cdots+\lambda_ny_n^2$
 求正交矩阵的步骤：
1. 求出矩阵的全部特征值
2. 由求出每个特征值对应的特征向量
3. 用施密特正交法将特征向量正交化单位化
4. 用第 3 步得到的向量构成正交矩阵
 **2. 可逆变换将二次型化为标准形 (配方)**（可逆变换包含正交变换）
可逆变换步骤: (1)配方 (2)得到变换关系$x=Cy$
结果:$x^{\mathrm{T}}Ax\xrightarrow{x=Qy}y^{\mathrm{T}}C^{\mathrm{T}}ACy=y^{\mathrm{T}}\Lambda y=d_1y_1^2+d_2y_2^2+\cdots+d_ny_n^2$
 (1)<u>配方法: 一次配方消除一个变量所有项, 后续配方时前一个变量不再存在。所以随着配方的进行, 变量越来越少，配方难度越来越低。</u>
$f=x^{\mathrm{T}}Ax=d_{1}(b_{11}x_{1}+...+b_{1n}x_{n})^{2}+d_{2}(b_{22}x_{2}+...+b_{2n}x_{n})^{2}+\cdots+d_{n}(b_{mn}x_{n})^{2}$
$=d_{1}y_{1}^{2}+d_{2}y_{2}^{2}+\cdots+d_{n}y_{n}^{2}$
 则有$\begin{pmatrix}y_1\\y_2\\...\\y_n\end{pmatrix}=\begin{pmatrix}b_{11}&b_{12}&...&b_{1n}\\0&b_{22}&...&b_{2n}\\...&...&...&...\\0&0&...&b_{nn}\end{pmatrix}\begin{pmatrix}x_1\\x_2\\...\\...\\x_n\end{pmatrix}$记为$y=Bx$反解出$x=Cy$其中$C=B^{-1}$
所以令$x=Cy$,可得$f=x^{\mathrm{T}}Ax=y^{\mathrm{T}}C^{\mathrm{T}}ACy=y^{\mathrm{T}}\Lambda y$
 (2)只有当$B$可逆时, 才有$x=Cy$(其中$C=B^{-1}$), 此时$x=Cy$才能成为可逆变换, 此时才不会改变标准形。
 (3)如果写成矩阵形式:$C^{\mathrm{T}}AC=\Lambda$。即配方法可求出可逆矩阵 C, 使得$C^{\mathrm{T}}AC=\Lambda$
 (4)二次型中无平方项时可用变量替换化出平方项后再配方
若只有混合项, 比如有$x_1x_2$, 则用代换$\begin{cases}x_1=y_1+y_2\\x_2=y_1-y_2\\x_3=y_3\end{cases}$
即有$x_ix_j$时, 可以用代换$\begin{cases}x_i=y_i+y_j\\x_j=y_i-y_j\\x_k=y_k\end{cases}$
 例题 1. 用配方法把二次型化为标准型, 并求所作变换:$\quad f(x_1,x_2,x_3)=2x_1x_2+4x_1x_3-x_2^2-8x_3^2$

【解】$f(x_{1},x_{2},x_{3})=2x_{1}x_{2}+4x_{1}x_{3}-x_{2}^{2}-8x_{3}^{2};$
$=-(x_{2}^{2}-2x_{1}x_{2}+x_{1}^{2})+x_{1}^{2}+4x_{1}x_{3}-8x_{3}^{2}$
$=-(x_{2}-x_{1})^{2}+(x_{1}+2x_{3})^{2}-12x_{3}^{2}$
令$\begin{cases}y_1=-x_1+x_2\\y_2=x_1+2x_3\\y_3=x_3\end{cases}$$\left.\Longrightarrow\left\{\begin{matrix}{x_{1}=y_{2}-2y_{3}}\\{x_{2}=y_{1}+y_{2}-2y_{3}}\\{x_{3}=y_{3}}\\\end{matrix}\right.\right.$由于$\begin{vmatrix}-1&1&0\\1&0&2\\0&0&1\end{vmatrix}\ne0$
所以上面交换为可逆变换. 得:$f(x_{1},x_{2},x_{3})=-y_{1}^{2}+y_{2}^{2}-12y_{3}^{2}.$

 【解】矩阵$A$对应的二次型为
$f\begin{pmatrix}x_1,x_2,x_3\end{pmatrix}=x_1^2+4x_1x_2+4x_2x_3$
$=\left(x_{1}^{2}+4x_{1}x_{2}+4x_{2}^{2}\right)-4x_{2}^{2}+4x_{2}x_{3}$
$=\left(x_{1}+2x_{2}\right)^{2}-\left(2x_{2}-x_{3}\right)^{2}+x_{3}^{2}$
 令$\begin{cases}y_1=x_1+2x_2,\\y_2=2x_2-x_3,\\y_3=x_3,\end{cases}$解得$\begin{cases}x_1=y_1-y_2-y_3,\\x_2=\dfrac{1}{2}\big(y_2+y_3\big),\\x_3=y_3,\end{cases}$即$\left.\left(\begin{array}{c}{x_{1}}\\{x_{2}}\\{x_{3}}\\\end{array}\right.\right)=\left(\begin{array}{ccc}{1}&{-1}&{-1}\\{0}&{\frac{1}{2}}&{\frac{1}{2}}\\{0}&{0}&{1}\\\end{array}\right)\left(\begin{array}{c}{y_{1}}\\{y_{2}}\\{y_{3}}\\\end{array}\right),$
 故可取可逆矩阵$C=\begin{pmatrix}1&-1&-1\\0&\dfrac{1}{2}&\dfrac{1}{2}\\0&0&1\end{pmatrix}$满足题意。注意本题答案不唯一。言之有理即可。
 **3. 矩阵三大关系的判别**
 （1）矩阵关系
1. A，B 等价：
表达式：$PAQ=B$，其中$P,Q$均为可逆矩阵
口诀：同型矩阵秩相等，则等价
关系：合同必等价，相似必等价
2. A，B 合同：
表达式：$C^{\mathrm{T}}AC=B$（其中$C$可逆）
口诀：具有相同的正负惯性指数，则合同
关系：合同必等价，<span style="background:#b1ffff">相似不一定合同，实对称矩阵相似必合同</span>
3. A，B 相似：
表达式：$P^{-1}AP=B$（其中$P$可逆）
口诀：实对称矩阵特征值相同，则相似
关系：相似必等价，相似不一定合同，实对称矩阵相似必合同 
4. <span style="background:#b1ffff">相似要求最高，但相似不一定合同，只有实对称时才有相似必合同</span>
5. <span style="background:#b1ffff">特征值相同不一定相似。可对角化且特征值相同，则一定相似</span> 
  
  （2）如何区分合同、相似、等价这一组晦涩的概念。
 物种与物种之间、矩阵与矩阵之间都存在相近关系。相似、合同、等价是描述两矩阵相近程度的词。"等价”是最弱的关系，仅要求两矩阵的秩相等（好比柯基和柯基拥有相同的 39 对染色体）；“合同”是较强的关系，还要求柯基的性别相同（好比特征值正负的个数相同），“相似”是更强的关系，要求特征值完全相同（好比两只柯基一模一样）。
 **4. 由初等变换建立方程判断两矩阵的关系**
方法：可通过初等变换建立方程，从而判断两矩阵的关系
1. 基本刻等矩阵的定义
$E_{ij}:$交换$E$第$i$行与第$j$行 (或交换第$i$列与第$j$列)所得的初等矩阵
$E_i(k)(k\neq0):E$的第$i$行 (或第$i$列)乘以非零常数所得的初等矩阵
$E_{ij}\left(k\right){:}$$E$的第$i$行的$k$倍加到第$j$行所得的初等矩阵
           $E$的第$j$列的$k$倍加到第$i$列所得的初等矩阵
2. 逆矩阵:$E_{ij}^{-1}=E_{ij};E_{i}^{-1}\left(k\right)=E_{i}\left(\frac{1}{k}\right);E_{ij}^{-1}\left(k\right)=E_{ij}\left(-k\right)$
3. 行列式:$\left|E_{ij}\right|=-1;\left|E_{i}\left(k\right)\right|=k\left(k\neq0\right);\left|E_{ij}\left(k\right)\right|=1$
 例题 1 设$A$是$n$阶方阵, 交换$A$的第$i,j$列后再交换第$i,j$行得到的矩阵记为$B$,则$A$和$B$（   ）
 (A) 等价但不相似
 (B) 相似但不合同
 (C) 相似、合同但不等价
 (D)相似、等价、合同
 【解】选 D。$B=E_{ij}AE_{ij}$,因为$E_{ij}^{-1}=E_{ij},E_{ij}^{\mathrm{T}}=E_{ij}$,所以$B=E_{ij}^{-1}AE_{ij},B=E_{ij}^{\mathrm{T}}AE_{ij}$。故$A$和$B$相似、等价、合同。
 **5. 判别正定矩阵的方法**
1.正定定义：若对任意$x\neq0$，都有$f=x^{\mathrm{T}}Ax>0$，则称二次型正定，$A$为正定矩阵
2.等价条件
$f(x)=x^{\mathrm{T}}Ax>0,\forall x\neq0$
$A$的各阶顺序主子式全大于零
$A$的所有特征值大于零
$A$的正惯性指数为$n$
 可逆阵$P$使$A=P^{\mathrm{T}}P($拆分为可逆阵, 也可理解为合同于单位阵)
 可逆阵$C$使$A=C^{\mathrm{T}}EC($合同于单位阵)
3.必要条件$:|A|{>}0,a_{ii}>0($<u>可用于快速判断矩阵是否正定</u>) 
4.三个正定: 设$A$正定, 则$A^{\mathrm{T}},A^{-1},A^*$均正定 (一人正定, 全家正定)
例题 1.下列各矩阵中是正定矩阵的是（   ）
$\text{(A)}\begin{bmatrix}1&1&1\\1&2&3\\1&3&6\end{bmatrix}\quad\text{(B)}\begin{bmatrix}1&1&1\\0&1&2\\0&0&-1\end{bmatrix}\quad\text{(C)}\begin{bmatrix}1&-1&1\\-1&1&2\\1&2&-8\end{bmatrix}\quad\text{(D)}\begin{bmatrix}2&3&4\\3&1&5\\4&5&6\end{bmatrix}$
【解】
方法1 对于选项 (A) ，$D_3=\begin{vmatrix}1&1&1\\1&2&3\\1&3&6\end{vmatrix}=1,D_2=\begin{vmatrix}1&1\\1&2\end{vmatrix}=1,D_1=1$
即(A)中短阵的各阶顺序主子式均大于零，故其为正定矩阵。
方法 2 （排除法）根据矩阵正定的充要条件，若矩阵为正定矩阵，则其各阶顺序主子式大于零. 四个选项中的 1 阶顺序主子式大于零，2 阶主子式中，选项（C）的 2 阶顺序主子式$D_2=\begin{vmatrix}1&-1\\-1&1\end{vmatrix}=0$，选项(D)的 2 阶顺序主子式$D_2=\begin{vmatrix}2&3\\3&1\end{vmatrix}=-7<0,$所以排除（C）、（D），而选项（B）的 3 阶顺序主子式为上三角行列式，即$D_3=\begin{vmatrix}1&1&1\\0&1&2\\0&0&-1\end{vmatrix}=-1<0$，故排除（B），所以选择（A）.
