# 版块 14 空间解析几何 (仅数一)

## A. 实用解题工具

### 第 1 组: 直线方程

直线方程有如下类型

1. 一般式方程: $\left\{  \begin{array}{ll} {A}_{1}x + {B}_{1}y + {C}_{1}x + {D}_{1} = 0 & \text{ 平面 }{\pi }_{1} \\  {A}_{2}x + {B}_{2}y + {C}_{2}x + {D}_{2} = 0 & \text{ 平面 }{\pi }_{2} \end{array}\right.$

2. 标准式方程: $\frac{x - {x}_{0}}{l} = \frac{y - {y}_{0}}{m} = \frac{z - {z}_{0}}{n}$

3. 两点式方程: $\frac{x - {x}_{1}}{{x}_{2} - {x}_{1}} = \frac{y - {y}_{1}}{{y}_{2} - {y}_{1}} = \frac{z - {z}_{1}}{{z}_{2} - {z}_{1}}$

4. 参数式方程: $\left\{  \begin{array}{lll} x = {x}_{0} + {lt} \\  y = {y}_{0} + {mt} \\  z = {z}_{0} + {nt} \end{array}\right.$

### 第 2 组: 直线平面位置

1. 平面间位置关系 : $\left\{  \begin{array}{llll} \text{ 定义 : }\left\{  \begin{array}{ll} {A}_{1}x + {B}_{1}y + {C}_{1}x + {D}_{1} = 0 & \text{ 平面 }{\pi }_{1} \\  {A}_{2}x + {B}_{2}y + {C}_{2}x + {D}_{2} = 0 & \text{ 平面 }{\pi }_{2} \end{array}\right. \\  \text{ 两平面平行 : } \Leftrightarrow  \frac{{A}_{1}}{{A}_{2}} = \frac{{B}_{1}}{{B}_{2}} = \frac{{C}_{1}}{{C}_{2}} \\  \text{ 两平面垂直 : } \Leftrightarrow  {A}_{1}{A}_{2} + {B}_{1}{B}_{2} + {C}_{1}{C}_{2} = 0 \\  \text{ 平面间夹角 }\theta  : \cos \theta  = \frac{{A}_{1}{A}_{2} + {B}_{1}{B}_{2} + {C}_{1}{C}_{2}}{\sqrt{{A}_{1}^{2} + {B}_{1}^{2} + {C}_{1}^{2}}\sqrt{{A}_{2}^{2} + {B}_{2}^{2} + {C}_{2}^{2}}} \end{array}\right.$

2. 平面与直线关系: $\left\{  \begin{array}{llll} \text{ 定义 : }\left\{  \begin{array}{ll} \text{ 直线 }L : \frac{x - {x}_{0}}{l} = \frac{y - {y}_{0}}{m} = \frac{z - {z}_{0}}{n} \\  \text{ 平面 }{\pi }_{1} : {A}_{1}x + {B}_{1}y + {C}_{1}z + {D}_{1} = 0 \end{array}\right. \\  \text{ 平行 } :  \Leftrightarrow  {Al} + {Bm} + {Cn} = 0 \\  \text{ 垂直 } :  \Leftrightarrow  \frac{A}{l} = \frac{B}{m} = \frac{C}{n} \\  \text{ 线面夹角 }\theta  : \sin \theta  = \frac {{Al} + {Bm} + {Cn}} {\sqrt {{A}^{2} + {B}^{2} + {C}^{2}} \sqrt {{l}^{2} + {m}^{2} + {n}^{2}} } \end{array}\right.$

3. 直线位置关系: $\left\{  \begin{array}{llll} \text{ 定义: }\left\{  \begin{array}{ll} {L}_{1} : \frac{x - {x}_{1}}{{l}_{1}} = \frac{y - {y}_{1}}{{m}_{1}} = \frac{z - {z}_{1}}{{n}_{1}} \\  {L}_{2} : \frac{x - {x}_{2}}{{l}_{2}} = \frac{y - {y}_{2}}{{m}_{2}} = \frac{z - {z}_{2}}{{n}_{2}} \end{array}\right. \\  \text{ 平行 } :  \Leftrightarrow  \frac{{l}_{1}}{{l}_{2}} = \frac{{m}_{1}}{{m}_{2}} = \frac{{n}_{1}}{{n}_{2}} \\  \text{ 垂直 } :  \Leftrightarrow  {l}_{1}{l}_{2} + {m}_{1}{m}_{2} + {n}_{1}{n}_{2} = 0 \\  \text{ 直线夹角 }\theta  : \cos \theta  = \frac{\left| {l}_{1}{l}_{2} + {m}_{1}{m}_{2} + {n}_{1}{n}_{2}\right| }{\sqrt{{l}_{1}^{2} + {m}_{1}^{2} + {n}_{1}^{2}}\sqrt{{l}_{2}^{2} + {m}_{2}^{2} + {n}_{2}^{2}}} \end{array}\right.$

### 第 3 组: 常见曲面

1. 球/椭球: $\left\{  \begin{array}{ll} \text{ 球 : }{x}^{2} + {y}^{2} + {z}^{2} = {a}^{2} \\  \text{ 椭球 : }\frac{{x}^{2}}{{a}^{2}} + \frac{{y}^{2}}{{b}^{2}} + \frac{{z}^{2}}{{c}^{2}} = 1 \end{array}\right.$

2. 锥面: ${z}^{2} = {x}^{2}$ , $y{y}^{2}\left\{  \begin{array}{ll} z = \sqrt{{x}^{2} + {y}^{2}} \\  z =  - \sqrt{{x}^{2} + {y}^{2}} \end{array}\right.$

3. 抛物面: $\left\{  \begin{array}{lll} \text{ 旋转抛物面: }z = {x}^{2} + {y}^{2} \\  \text{ 椭圆抛物面: }z = \frac{{x}^{2}}{{a}^{2}} + \frac{{y}^{2}}{{b}^{2}} \\  \text{ 双曲抛物面: }z = \frac{{x}^{2}}{{a}^{2}} - \frac{{y}^{2}}{{b}^{2}} \end{array}\right.$

4. 单叶/双叶双曲面: $\left\{  \begin{array}{ll} \text{ 单. 叶 } : \frac{{x}^{2}}{{a}^{2}} + \frac{{y}^{2}}{{b}^{2}} - \frac{{z}^{2}}{{c}^{2}} = 1 \\  \text{ 双叶 } : \frac{{x}^{2}}{{a}^{2}} - \frac{{y}^{2}}{{b}^{2}} - \frac{{z}^{2}}{{c}^{2}} = 1 \end{array}\right.$

5. 柱面: $\left\{  \begin{array}{ll} \text{ 柱面通式: }F\left( {x,y}\right)  = 0\text{ 表示母线平行于 }z \\  \text{ 常见柱面: 圆柱面: }{x}^{2} + {y}^{2} = {a}^{2} \end{array}\right.$  
  
 **1.求空间曲线的在坐标面上的投影曲线**

方法: 空间曲线 $\Gamma$ 的方程写成两个曲面联立的形式 $\left\{  \begin{array}{ll} F\left( {x,y,z}\right)  = 0 \\  G\left( {x,y,z}\right)  = 0 \end{array}\right.$ 。若要求在坐标面 ${xOy}$ 上的投影,先在方程 $\left\{  \begin{array}{ll} F\left( {x,y,z}\right)  = 0 \\  G\left( {x,y,z}\right)  = 0 \end{array}\right.$ 中消去变量 $z$ ,得到 $f\left( {x,y}\right)  = 0$ 。再写成 $\left\{  \begin{array}{ll} f\left( {x,y}\right)  = 0 \\  z = 0 \end{array}\right.$ 的形式。其他坐标面投影类似。

 **2.求空间曲线旋转所得的曲面方程**

1. 求解旋转曲面的思路: 空间曲线上的任意一点, 绕轴旋转均得出一个圆, 圆半径为点到轴的距离, 圆心在旋转轴上。

2. 特别地, $f\left( {x,z}\right)  = 0$ 绕 $z$ 轴旋转,得到 $f\left( {\pm \sqrt{{x}^{2} + {y}^{2}},z}\right)  = 0$

例题 1. 空间直线 $x = y = z$ 绕 $z$ 轴旋转得到的曲面方程为___

【解】设空间曲线上任意点 $\left( {{x}_{0},{y}_{0},{z}_{0}}\right)$ 满足 ${x}_{0} = {y}_{0} = {z}_{0}$ ,绕 $z$ 轴旋转,所得出的点的坐标始终满足 $z = {z}_{0}$ ,

且 ${x}^{2} + {y}^{2} = {x}_{0}^{2} + {y}_{0}^{2}$ 。整理两式,消除 ${x}_{0},{y}_{0},{z}_{0}$ 可得到 ${x}^{2} + {y}^{2} = {x}_{0}^{2} + {y}_{0}^{2} = {z}_{0}^{2} + {z}_{0}^{2} = {2z}$

即, ${x}^{2} + {y}^{2} = {2z}$ 为所求。

 **3.利用距离公式求解相关问题**

1. 点到平面距离公式: $d = \frac{\left| A{x}_{0} + B{y}_{0} + C{z}_{0} + D\right| }{\sqrt{{A}^{2} + {B}^{2} + {C}^{2}}}$

2. 点到直线距离公式: $d = \frac{\left| \overrightarrow{{M}_{1}{M}_{0}} \times  \overrightarrow{{M}_{1}P}\right| }{\overrightarrow{{M}_{1}P}} = \frac{\begin{Vmatrix} \overrightarrow{i} & \overrightarrow{j} & \overrightarrow{k} \\  {x}_{0} - {x}_{1} & {y}_{0} - {y}_{1} & {z}_{0} - {z}_{1} \\  l & m & n \end{Vmatrix}}{\sqrt{{l}^{2} + {m}^{2} + {n}^{2}}}\left( {{L}_{1} : \frac{x - {x}_{1}}{l} = \frac{y - {y}_{1}}{m} = \frac{z - {z}_{1}}{n}}\right)$ 
 **4.利用平面束方法解题**
 平面束方程:
通过直线 $l : \left\{  \begin{array}{ll} {A}_{1}x + {B}_{1}y + {C}_{1}z + {D}_{1} = 0, \\  {A}_{2}x + {B}_{2}y + {C}_{2}z + {D}_{2} = 0 \end{array}\right.$ 的平面束方程为
${A}_{1}x + {B}_{1}y + {C}_{1}z + {D}_{1} + \lambda \left( {{A}_{2}x + {B}_{2}y + {C}_{2}z + {D}_{2}}\right)  = 0$

其中平面 ${A}_{2}x + {B}_{2}y + {C}_{2}z + {D}_{2} = 0$ 不在平面束方程内.
