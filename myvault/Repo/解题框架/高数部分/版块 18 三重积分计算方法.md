# 版块 18 三重积分计算方法
## A. 实用解题工具

### 第 1 组: 三重积分解题框架

## B. 核心问题及解题方法总结

 **1.三重积分简化

1. 若积分区域关于 ${yOz}$ 坐标面对称,被积函数具有奇偶性,则

$$
{\iiint }_{\Omega }f\left( {x,y,z}\right) {dV} = \left\{  \begin{array}{ll} 0,f\text{ 是 }x\text{ 的奇函数 } \\  {\iiint }_{\Omega }f\left( {x,y,z}\right) {dV},f\text{ 是 }x\text{ 的偶函数,}{\Omega }_{1}\text{ 是 }\Omega \text{ 前半部分 } \end{array}\right.
$$

2. 同理。 $f$ 是 $y$ 的奇偶函数或 $f$ 是 $z$ 的奇偶函数时,也有类似的性质。

3. 轮换对称 $\left\{  \begin{array}{ll} \text{ 若积分区域关于 }x,y\text{ 轮换对称 } \\  \text{ 则 }{\iiint }_{\Omega }f\left( {x,y,z}\right) \mathrm{d}V = {\iiint }_{\Omega }f\left( {y,x,z}\right) \mathrm{d}V \end{array}\right.$

【笔记】轮换对称对函数 $f\left( {x,y,z}\right)$ 无要求

 **2.三重积分直角坐标

直角坐标 $\left\{  \begin{array}{ll} {\iiint }_{\Omega }f\left( {x,y,z}\right) \mathrm{d}V = {\int }_{a}^{b}{dx}{\int }_{{y}_{1}\left( x\right) }^{{y}_{2}\left( y\right) }{dy}{\int }_{{z}_{1}\left( {x,y}\right) }^{{z}_{2}\left( {x,y}\right) }f\left( {x,y,z}\right) {dz} \\  \text{ 其他积分展开次序同理 } \end{array}\right.$ :

 **3.三重积分柱坐标

1. 公式: ${\iiint }_{\Omega }f\left( {x,y,z}\right) \mathrm{d}V = {\int }_{{\theta }_{1}}^{{\theta }_{2}}{d\theta }{\int }_{{r}_{1}\left( \theta \right) }^{{r}_{2}\left( \theta \right) }{rdr}{\int }_{{z}_{1}\left( {x,y}\right) }^{{z}_{2}\left( {x,y}\right) }f\left( {x,y,z}\right) {dz}$

2. 适用情况：投影区域边界含圆或圆弧

3. 柱坐标与直角坐标的关系为 $\left\{  \begin{matrix} x = r\cos \theta , & 0 \leq  r <  + \infty , \\  y = r\sin \theta , & 0 \leq  \theta  \leq  {2\pi }, \\  z = z, &  - \infty  < z <  + \infty . \end{matrix}\right.$

例题 1. 计算 ${\iiint }_{\Omega }z{e}^{{x}^{2} + {y}^{2}}{dV}$ ,其中 $\Omega$ 是由曲面 $z = \sqrt{{x}^{2} + {y}^{2}}$ 以及平面 $z = 2$ 所围成的区域.

【解】 ${\iiint }_{\Omega }z{e}^{{x}^{2} + {y}^{2}}{dV} = {\iiint }_{\Omega }z{e}^{{r}^{2}}{rdrd\theta dz} = {\int }_{0}^{2\pi }{d\theta }{\int }_{0}^{2}{e}^{{r}^{2}}{rdr}{\int }_{r}^{2}{zdz}$

$$
= {\int }_{0}^{2\pi }\left\lbrack  {{\int }_{0}^{2}{e}^{{r}^{2}} \cdot  \frac{1}{2}\left( {{2}^{2} - {r}^{2}}\right) {rdr}}\right\rbrack  {d\theta } = \pi {\int }_{0}^{2}{e}^{{r}^{2}}\left( {{2}^{2} - {r}^{2}}\right) {rdr} = \frac{\pi }{2}\left( {{e}^{4} - 5}\right) .
$$
