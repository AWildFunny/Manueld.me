 版块 19 曲线积分计算 (仅数一)

## B. 核心问题及解题方法总结

 **1.一型曲线的简化

简化方式 1: 奇对称 $\left\{  \begin{array}{ll} \text{ 若 }L\text{ 关于 }x\text{ 轴 }\left( {\text{ 或 }y\text{ 轴 }}\right) \text{ 对称,}f\left( {x,y}\right) \text{ 为 }y\left( {\text{ 或 }x}\right) \text{ 的奇函数 } \\  \text{ 则 }{\int }_{L}f\left( {x,y}\right) \mathrm{d}s = 0 \end{array}\right.$

简化方式 2: 偶对称 $\left\{  \begin{array}{ll} \text{ 若 }L\text{ 关于 }x\left( {\text{ 或 }y}\right) \text{ 对称,}f\left( {x,y}\right) \text{ 为 }x\left( {\text{ 或 }y}\right) \text{ 的偶函数 } \\  \text{ 则 }{\int }_{L}f\left( {x,y}\right) \mathrm{d}s = 2{\int }_{{L}_{1}}f\left( {x,y}\right) \mathrm{d}s\text{ (其中 }{L}_{1}\text{ 为 }L\text{ 的一半) } \end{array}\right.$

简化方式 3: 轮换对称 $\left\{  \begin{array}{ll} \text{ 若 }L\text{ 关于 }y = x\text{ 对称,}f\left( {x,y}\right) \\  \text{ 则 }{\int }_{L}f\left( {x,y}\right) \mathrm{d}s = {\int }_{L}f\left( {y,x}\right) \mathrm{d}s \end{array}\right.$

简化方式 4: 方程简化: 曲线 $L$ 的方程可直接代入 ${\int }_{L}f\left( {x,y}\right) \mathrm{d}s$ 的被积函数

 例题1. 设 $\Gamma  : \left\{  \begin{array}{ll} {x}^{2} + {y}^{2} + {z}^{2} = 4, \\  x + y + z = 0, \end{array}\right.$ 则 ${\oint }_{\Gamma }{x}^{2}{ds} =$  
 
 【解】 ${\oint }_{\Gamma }{x}^{2}{ds} = \frac{1}{3}{\oint }_{\Gamma }\left( {{x}^{2} + {y}^{2} + {z}^{2}}\right) {ds} = \frac{4}{3}{\oint }_{\Gamma }{ds} = \frac{4}{3} \cdot  {2\pi } \cdot  2 = \frac{16}{3}\pi$ ,

 **2. 定积分计算一型曲线

计算: 化为定积分 ${\int }_{L}f\left( {x,y}\right) \mathrm{d}s = {\int }_{a}^{b}f\left( {x\left( t\right) ,y\left( t\right) }\right) \sqrt{{\left( {x}_{t}^{\prime }\right) }^{2} + {\left( {y}_{t}^{\prime }\right) }^{2}}\mathrm{\;d}t$

 **3. 简化与计算二型曲线积分

1. 方程简化: 曲线 $L$ 的方程可直接代入 ${\int }_{L}P\mathrm{\;d}x + Q\mathrm{\;d}y$ 的被积函数中

2. 定积分计算: ${\int }_{L}P\mathrm{\;d}x + Q\mathrm{\;d}y = {\int }_{a}^{b}\left\lbrack  {P\left( {x\left( t\right) ,y\left( t\right) }\right) {x}_{t}^{\prime } + Q\left( {x\left( t\right) ,y\left( t\right) }\right) {y}_{t}^{\prime }}\right\rbrack  \mathrm{d}t$

 **4. 利用格林公式计算二型曲线积分

1. 格林公式

封闭正向: 格林公式 ${\oint }_{L}P\mathrm{\;d}x + Q\mathrm{\;d}y = \iint \left( {\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}}\right) \mathrm{d}x\mathrm{\;d}y$

2. 非封闭的处理方法

非封闭正向 $\left\{  \begin{array}{ll} \text{ 补线: }{\int }_{L}P\mathrm{\;d}x + Q\mathrm{\;d}y = {\oint }_{L + {L}_{1}}P\mathrm{\;d}x + Q\mathrm{\;d}y - {\int }_{{L}_{1}}P\mathrm{\;d}x + Q\mathrm{\;d}y \\  \text{ 其中,格林公式计算 }{\oint }_{L + {L}_{1}}P\mathrm{\;d}x + Q\mathrm{\;d}y\text{; 定积分计算}{\int }_{{L}_{1}}P\mathrm{\;d}x + Q\mathrm{\;d}y \end{array}\right.$

 **5. 挖洞法计算二型曲线积分

非单连通区域上二型曲线积分的处理方法: 挖洞

非单连通 $\left\{  \begin{array}{ll} \text{ 补线挖洞: }{\oint }_{L}P\mathrm{\;d}x + Q\mathrm{\;d}y = {\oint }_{L + {L}_{1}}P\mathrm{\;d}x + Q\mathrm{\;d}y - {\oint }_{{L}_{1}}P\mathrm{\;d}x + Q\mathrm{\;d}y \\  \text{ 其中,格林公式计算 }{\oint }_{L + {L}_{1}}P\mathrm{\;d}x + Q\mathrm{\;d}y\text{; 定积分或格林计算}{\oint }_{{L}_{1}}P\mathrm{\;d}x + Q\mathrm{\;d}y \end{array}\right.$

例题 1. 计算曲线积分 ${\oint }_{L}\frac{\left( {x - y}\right) {dx} + \left( {x + {4y}}\right) {dy}}{{x}^{2} + 4{y}^{2}}$ ,其中

（1） $L$ 为曲线 $\left| x\right|  + \left| y\right|  = 1$ 的逆时针方向。

（2） $L$ 为曲线 ${\left( x - 2\right) }^{2} + {\left( y - 2\right) }^{2} = 1$ 的逆时针方向。


【解】 $P = \frac{x - y}{{x}^{2} + 4{y}^{2}},Q = \frac{x + {4y}}{{x}^{2} + 4{y}^{2}},\frac{\partial Q}{\partial x} = \frac{\partial P}{\partial y}$ ,

（1）不能直接使用格林公式。令 $C : {x}^{2} + 4{y}^{2} = {r}^{2}\left( {r > 0}\right)$ 逆时针且 $C$ 在曲线 $L$ 内,曲线 $\mathrm{C}$ 围成区域 $D$

则有 ${\oint }_{L}\frac{\left( {x - y}\right) {dx} + \left( {x + {4y}}\right) {dy}}{{x}^{2} + 4{y}^{2}}$

$$
= {\oint }_{C}\frac{\left( {x - y}\right) {dx} + \left( {x + {4y}}\right) {dy}}{{x}^{2} + 4{y}^{2}} = \frac{1}{{r}^{2}}{\oint }_{C}\left( {x - y}\right) {dx} + \left( {x + {4y}}\right) {dy}
$$

$$
= \frac{1}{{r}^{2}}{\iint }_{D}{2d\sigma } = 2 \times  \frac{1}{{r}^{2}} \times  \pi  \times  r \times  \frac{r}{2} = \pi
$$

（2）曲线 $L$ 围成区域 ${D}_{2}$ ,不包含零点。可直接使用格林公式。

$$
{\oint }_{L}\frac{\left( {x - y}\right) {dx} + \left( {x + {4y}}\right) {dy}}{{x}^{2} + 4{y}^{2}} = {\iint }_{{D}_{2}}\left( {\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}}\right) \mathrm{d}x\mathrm{\;d}y = 0
$$

 **6. 与路径无关的判别与计算

1. 与路径无关的等价条件 $\left\{  \begin{array}{llll} \text{ ① }\frac{\partial Q}{\partial x} = \frac{\partial P}{\partial y} \\  \text{ ② }{\oint }_{L}P\mathrm{\;d}x + Q\mathrm{\;d}y = 0 \\  \text{ ③ }P\mathrm{\;d}x + Q\mathrm{\;d}y = \mathrm{d}u \\  \text{ 注意: 在单连通区域上. } \end{array}\right.$

2. 与路径无关时的计算方法

方法 1: 凑原函数: $P\mathrm{\;d}x + Q\mathrm{\;d}y = \mathrm{d}u \Rightarrow  {\int }_{\left( {x}_{1},{y}_{1}\right) }^{\left( {x}_{2},{y}_{2}\right) }P\mathrm{\;d}x + Q\mathrm{\;d}y = {\left. u\left( x,y\right) \right| }_{\left( {x}_{1},{y}_{1}\right) }^{\left( {x}_{2},{y}_{2}\right) }$

方法 2: 折线计算: $\left\{  \begin{array}{ll} {\int }_{\left( {x}_{1},{y}_{1}\right) }^{\left( {x}_{2},{y}_{2}\right) }P\mathrm{\;d}x + Q\mathrm{\;d}y = {\int }_{{x}_{1}}^{{x}_{2}}P\left( {x,{y}_{1}}\right) \mathrm{d}x + {\int }_{{y}_{1}}^{{y}_{2}}Q\left( {{x}_{2},y}\right) \mathrm{d}y \\  {\int }_{\left( {x}_{1},{y}_{1}\right) }^{\left( {x}_{2},{y}_{2}\right) }P\mathrm{\;d}x + Q\mathrm{\;d}y = {\int }_{{y}_{1}}^{{y}_{2}}P\left( {{x}_{1},y}\right) \mathrm{d}y + {\int }_{{x}_{1}}^{{x}_{2}}Q\left( {x,{y}_{2}}\right) \mathrm{d}x \end{array}\right.$

 例题 1. 设 $g\left( u\right)$ 为连续可微函数,若曲线积分 ${\int }_{C}y\left( {{e}^{x} + {2g}\left( x\right) }\right) {dx} - g\left( x\right) {dy}$ 与路径无关,且 $g\left( 0\right)  = 1$ ,

 求 ${\int }_{\left( 0,0\right) }^{\left( 1,1\right) }y\left( {{e}^{x} + {2g}\left( x\right) }\right) {dx} - g\left( x\right) {dy}$ .

【解】记 $P = y\left( {{e}^{x} + {2g}\left( x\right) }\right) ,Q =  - g\left( x\right)$ . 由 ${P}_{y} = {Q}_{x}$ 得 ${e}^{x} + {2g}\left( x\right)  =  - {g}^{\prime }\left( x\right)$ . 此方程满足  $g\left( 0\right)  = 1$ 的解为 $g\left( x\right)  =  - \frac{1}{3}\left( {{e}^{x} - 4{e}^{-{2x}}}\right)$ .

于是 ${\int }_{\left( 0,0\right) }^{\left( 1,1\right) }y\left( {{e}^{x} + {2g}\left( x\right) }\right) {dx} - g\left( x\right) {dy} =  - {yg}\left( x\right) \left( {{}_{\left( 0,0\right) }^{\left( 1,1\right) } =  - g\left( 1\right)  =  - \frac{4}{3}{e}^{-2} + \frac{1}{3}e}\right.$ .

 **7. 定积分计算第二型空间曲线

1. 二型空间曲线的方程简化: 可代入曲线方程化简被积函数。

2. 二型空间曲线积分计算: 
化为定积分: ${\int }_{L}P\mathrm{\;d}x + Q\mathrm{\;d}y + R\mathrm{\;d}z = {\int }_{a}^{b}\left\lbrack  {P\left( {x\left( t\right) ,y\left( t\right) ,z\left( t\right) }\right) {x}_{t}^{\prime } + Q{y}_{t}^{\prime } + R{z}_{t}^{\prime }}\right\rbrack  \mathrm{d}t$

 **8. 利用斯托克斯公式计算空间二型曲线积分

写法1: ${\int }_{\Gamma }{Pdx} + {Qdy} + {Rdz} = {\iint }_{S}\left| \begin{matrix} {dydz} & {dzdx} & {dxdy} \\  \frac{\partial }{\partial x} & \frac{\partial }{\partial y} & \frac{\partial }{\partial z} \\  P & Q & R \end{matrix}\right|$ (方向遵循右手定则)

写法 2: ${\int }_{\Gamma }{Pdx} + {Qdy} + {Rdz} = {\iint }_{S}\left| \begin{array}{lll} \cos \alpha & \cos \beta & \cos \gamma \\  \frac{\partial }{\partial x} & \frac{\partial }{\partial y} & \frac{\partial }{\partial z} \\  P & Q & R \end{array}\right| {dS}$

说明 $\left\{  \begin{array}{lll} \text{ 右手定则: 右手四指指向曲线方向,大拇指指向为曲面方向 } \\  \text{ 写法 }2\text{ 中 }\left( {\cos \alpha ,\cos \beta ,\cos \gamma }\right) \text{ 是曲面 }S\text{ 在点 }\left( {x,y,z}\right) \text{ 的单位法向量 } \\  \text{ 通过曲线的任何曲面均可,但要考虑计算上的难度 } \end{array}\right.$
