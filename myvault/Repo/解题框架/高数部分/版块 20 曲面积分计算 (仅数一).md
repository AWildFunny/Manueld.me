# 版块 20 曲面积分计算 (仅数一)

## B. 核心问题及解题方法总结

 **1. 简化一型曲面积分

方法 1: 奇偶对称简化

奇对称 $\left\{  \begin{array}{ll} \text{ 区域关于 }y\text{ oz }\text{ 面对称,} \\  \text{ 则 }{\iint }_{\sum }f\left( {x,y,z}\right) \mathrm{d}S = 0 \end{array}\right.$

偶对称 $\left\{  \begin{array}{ll} \text{ 区域关于 }y \geq  0\text{ 的对称,被积函数为 }x\text{的偶函数 } \\  \text{ 则 }{\iint }_{\sum }f\left( {x,y,z}\right) \mathrm{d}S = 2{\iint }_{{\sum }_{1}}f\left( {x,y,z}\right) \mathrm{d}S\text{,其中 }{\sum }_{1}\text{ 为 }\sum \text{ 的一半 } \end{array}\right.$

方法 2: 轮换对称简化

轮换对称 $\left\{  \begin{array}{ll} \text{ 积分区域关于 }x,y\text{ 轮换对称 } \\  {\iint }_{\sum }f\left( {x,y,z}\right) \mathrm{d}S = {\iint }_{\sum }f\left( {y,x,z}\right) \mathrm{d}S \end{array}\right.$

方法 3: 方程简化

方程简化: 可代入曲面方程化简被积函数

 **2. 计算第一型曲面积分

1. 运用二重积分计算: ${\iint }_{\sum }f\left( {x,y,z}\right) \mathrm{d}S = {\iint }_{{D}_{xy}}f\left\lbrack  {x,y,z\left( {x,y}\right) }\right\rbrack  \sqrt{1 + {z}_{x}^{\prime 2} + {z}_{y}^{\prime 2}}\mathrm{\;d}x\mathrm{\;d}y$

2. 应用。

① 计算曲面面积: $S = {\iint }_{\sum }\mathrm{d}S$

② 计算曲面质量: $m = {\iint }_{\sum }\mu \left( {x,y,z}\right) \mathrm{d}S = {\iint }_{{D}_{xy}}\mu \left\lbrack  {x,y,z\left( {x,y}\right) }\right\rbrack  \sqrt{1 + {z}_{x}^{\prime 2} + {z}_{y}^{\prime 2}}\mathrm{\;d}x\mathrm{\;d}y$

例题 1. 设 $\sum$ 是圆锥面 $z = \sqrt{{x}^{2} + {y}^{2}}$ 介于平面 $z = 0$ 与 $z = 1$ 之间的部分,则 ${\iint }_{\sum }\frac{\mathrm{d}S}{z} =$

【解】曲面 $\sum$ 在 ${xOy}$ 面上的投影区域 ${D}_{xy} : {x}^{2} + {y}^{2} \leq  1$ ,在 $\sum$ 上 $z = \sqrt{{x}^{2} + {y}^{2}}$ ,

所以 $\sqrt{1 + {z}_{x}^{\prime 2} + {z}_{y}^{\prime 2}} = \sqrt{1 + {\left( \frac{x}{\sqrt{{x}^{2} + {y}^{2}}}\right) }^{2} + {\left( \frac{y}{\sqrt{{x}^{2} + {y}^{2}}}\right) }^{2}} = \sqrt{2}$ ,

因此 ${\iint }_{\sum }\frac{\mathrm{d}S}{z} = {\iint }_{{D}_{xy}}\frac{1}{\sqrt{{x}^{2} + {y}^{2}}}\sqrt{2}\mathrm{\;d}x\mathrm{\;d}y = \sqrt{2}{\int }_{0}^{2\pi }\mathrm{d}\theta {\int }_{0}^{1}\frac{1}{\rho }\rho \mathrm{d}\rho  = 2\sqrt{2}\pi$ .

 **3. 简化与计算第二型曲面积分

1. 计算前先简化: 方程简化: 可代入曲面方程化简被积函数

2. 利用 $\rightarrow$ 重积分计算 $\left\{  \begin{array}{lll} {\iint }_{\sum }P\left( {x,y,z}\right) \mathrm{d}y\mathrm{\;d}z =  \pm  {\iint }_{{D}_{xy}}P\left\lbrack  {x\left( {y,z}\right) ,y,z}\right\rbrack  \mathrm{d}y\mathrm{\;d}z \\  {\iint }_{\sum }Q\left( {x,y,z}\right) \mathrm{d}x\mathrm{\;d}z =  \pm  {\iint }_{{D}_{xy}}Q\left\lbrack  {x,y\left( {x,z}\right) ,z}\right\rbrack  \mathrm{d}x\mathrm{\;d}z \\  {\iint }_{\sum }R\left( {x,y,z}\right) \mathrm{d}x\mathrm{\;d}y =  \pm  {\iint }_{{D}_{xy}}R\left\lbrack  {x,y,z\left( {x,y}\right) }\right\rbrack  \mathrm{d}x\mathrm{\;d}y \end{array}\right.$

 **4. 利用高斯公式计算曲面积分

1. 封闭外侧且单连通曲面的高斯公式:

封闭外侧: $${} \mathop{{\int\!\!\!\!\!\int}\mkern-21mu \bigcirc}\limits_\Sigma {Pdydz} + {Qdzdx} + {Rdxdy} = {\iiint }_{\Omega }\left( {\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}}\right) {dV}$$
2. 非封闭的处理方法

非封闭: $\left\{  \begin{array}{ll} \text{ 补面 } : {\iint }_{\sum }{Pdydz} + {Qdzdx} + {Rdxdy} = \left( {{\iint }_{\sum } + {\iint }_{{\sum }_{1}}}\right)  - {\iint }_{{\sum }_{1}} \\  \text{ 其中,运用高斯公式求 }{\iint }_{\sum } + {\iint }_{{\sum }_{1}}\text{,运用二重积分求 }{\int }_{\sum } \end{array}\right.$

3. 非单连通的处理方法

$$\mathop{{\int\!\!\!\!\!\int}\mkern-21mu \bigcirc}\limits_\Sigma  {xdydz} + {ydzdx} + {zdxdy} = {\iiint }_{\Omega }\left( {1 + 1 + 1}\right) {dxdydz}= $$
$$3{\int }_{0}^{2\pi }{d\theta }{\int }_{0}^{\frac{R}{2}}{rdr}{\int }_{\sqrt{3}r}^{\sqrt{{R}^{2} - {r}^{2}}}{dz} = {6\pi }{\int }_{0}^{\frac{R}{2}}r\left( {\sqrt{{R}^{2} - {r}^{2}} - \sqrt{3}r}\right) {dr}

 **5. 利用合一投影法求解

1. 两类曲面积分的关系

$$
{\iint }_{\sum }{Pdydz} + {Qdzdx} + {Rdxdy} = {\iint }_{\sum }\left( {P\cos \alpha  + Q\cos \beta  + R\cos \gamma }\right) {dS}
$$

【笔记】法向量 $\left\{  \begin{array}{lll} \cos \alpha  = \frac{\pm {z}_{x}^{\prime }}{\sqrt{1 + {\left( {z}_{x}^{\prime }\right) }^{2} + {\left( {z}_{y}^{\prime }\right) }^{2}}} \\  \cos \beta  = \frac{\pm {z}_{y}^{\prime }}{\sqrt{1 + {\left( {z}_{x}^{\prime }\right) }^{2} + {\left( {z}_{y}^{\prime }\right) }^{2}}} \\  \cos \gamma  = \frac{\mp 1}{\sqrt{1 + {\left( {z}_{x}^{\prime }\right) }^{2} + {\left( {z}_{y}^{\prime }\right) }^{2}}} \end{array}\right.$ 正负取决于曲面方向

2. 合一投影法

$$
{\iint }_{\sum }{Pdydz} + {Qdzdx} + {Rdxdy} = {\iint }_{\sum }\left\lbrack  {P\left( {-\frac{\partial z}{\partial x}}\right)  + Q\left( {-\frac{\partial z}{\partial y}}\right)  + R}\right\rbrack  {dxdy}
$$

合一投影的转换公式: $\left\{  \begin{array}{ll} {dydz} =  - \frac{\partial z}{\partial x}{dxdy} \\  {dzdx} =  - \frac{\partial z}{\partial y}{dxdy} \end{array}\right.$

例题 1. 设 $f\left( {x,y,z}\right)$ 是连续函数, $\sum$ 是平面 $x - y + z - 1 = 0$ 在第四卦限部分的上侧,计算

$$
{\iint }_{\sum }\left\lbrack  {f\left( {x,y,z}\right)  + x}\right\rbrack  {dydz} + \left\lbrack  {{2f}\left( {x,y,z}\right)  + y}\right\rbrack  {dzdx} + \left\lbrack  {f\left( {x,y,z}\right)  + z}\right\rbrack  {dxdy}.
$$

【解】 $\sum$ 的法向量为 $n = \{ 1, - 1,1\}$ ,方向余弦为 $\cos \alpha  = \frac{1}{\sqrt{3}},\cos \beta  =  - \frac{1}{\sqrt{3}},\cos \gamma  = \frac{1}{\sqrt{3}}$ ,根据两类曲面积分之间的关系有

$$
{\iint }_{\sum }\left\lbrack  {f\left( {x,y,z}\right)  + x}\right\rbrack  {dydz} + \left\lbrack  {{2f}\left( {x,y,z}\right)  + y}\right\rbrack  {dzdx} + \left\lbrack  {f\left( {x,y,z}\right)  + z}\right\rbrack  {dxdy}
$$

$$
= \frac{1}{\sqrt{3}}{\iint }_{\sum }\{ \left\lbrack  {f\left( {x,y,z}\right)  + x}\right\rbrack   - \left\lbrack  {{2f}\left( {x,y,z}\right)  + y}\right\rbrack   + \left\lbrack  {f\left( {x,y,z}\right)  + z}\right\rbrack  \} {ds}
$$

$$
= \frac{1}{\sqrt{3}}{\iint }_{\sum }\left( {x - y + z}\right) {ds} = \frac{1}{\sqrt{3}}{\iint }_{\sum }{ds} = \frac{1}{\sqrt{3}} \times  \frac{1}{2} \times  \sqrt{2} \times  \sqrt{2} \times  \sin \frac{\pi }{3} = \frac{1}{2}
$$

【说明】本题也可根据合一投影法求解。

 **6. 积分与曲面无关的情况

方法: 曲面积分与曲面无关,则有 $\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z} = 0$  

 **7.计算梯度与方向导数

1. 梯度: $u = f\left( {x,y,z}\right)$ 的梯度为 $\left( {{f}_{x}^{\prime },{f}_{y}^{\prime },{f}_{z}^{\prime }}\right)$

2. 方向导数定义

定义: ${\left. \frac{\partial f}{\partial l}\right| }_{\left( {x}_{0},{y}_{0},{z}_{0}\right) } = \mathop{\lim }\limits_{{t \rightarrow  {0}^{ + }}}\frac{f\left( {{x}_{0} + t\cos \alpha ,{y}_{0} + t\cos \beta ,{z}_{0} + t\cos \gamma }\right)  - f\left( {{x}_{0},{y}_{0},{z}_{0}}\right) }{t}$

3. 方向导数的计算公式

$\frac{\partial f}{\partial l} = \left( {\cos \alpha ,\cos \beta ,\cos \gamma }\right)  \cdot  \left( {{f}_{x}^{\prime },{f}_{y}^{\prime },{f}_{z}^{\prime }}\right) \;$ 【注意】本公式仅在 $f$ 可微时才成立。

4. 方向导数最值

方向导数最值 $\left\{  \begin{array}{ll} \text{ 最大方向导数: }\left| {\operatorname{grad}f\left( {{x}_{0},{y}_{0},{z}_{0}}\right) }\right| \\  \text{ 最小方向导数: } - \left| {\operatorname{grad}f\left( {{x}_{0},{y}_{0},{z}_{0}}\right) }\right|  \end{array}\right.$

 **8.计算切向量与法向量
 
1. 曲线 $\left\{  \begin{array}{lll} x = x\left( t\right) \\  y = y\left( t\right) \\  z = z\left( t\right)  \end{array}\right.$ 的切向量为 $\left( {{x}^{\prime }\left( t\right) ,{y}^{\prime }\left( t\right) ,{z}^{\prime }\left( t\right) }\right)$ ,由此可写出:

① 切线方程: $\frac{x - {x}_{0}}{{x}^{\prime }\left( {t}_{0}\right) } = \frac{y - {y}_{0}}{{y}^{\prime }\left( {t}_{0}\right) } = \frac{z - {z}_{0}}{{z}^{\prime }\left( {t}_{0}\right) }$

② 法平面方程: ${x}^{\prime }\left( {t}_{0}\right) \left( {x - {x}_{0}}\right)  + {y}^{\prime }\left( {t}_{0}\right) \left( {y - {y}_{0}}\right)  + {z}^{\prime }\left( {t}_{0}\right) \left( {z - {z}_{0}}\right)  = 0$

2. 曲面 $f\left( {x,y,z}\right)  = 0$ 的法向量为 $\left( {{f}_{x}^{\prime },{f}_{y}^{\prime },{f}_{z}^{\prime }}\right)$ ,由此可写出

① 切平面方程: ${f}_{x}^{\prime }\left( {x - {x}_{0}}\right)  + {f}_{y}^{\prime }\left( {y - {y}_{0}}\right)  + {f}_{z}^{\prime }\left( {z - {z}_{0}}\right)  = 0$

② 法线方程: $\frac{x - {x}_{0}}{{f}_{x}^{\prime }\left( {{x}_{0},{y}_{0},{z}_{0}}\right) } = \frac{y - {y}_{0}}{{f}_{y}^{\prime }\left( {{x}_{0},{y}_{0},{z}_{0}}\right) } = \frac{z - {z}_{0}}{{f}_{z}^{\prime }\left( {{x}_{0},{y}_{0},{z}_{0}}\right) }$
