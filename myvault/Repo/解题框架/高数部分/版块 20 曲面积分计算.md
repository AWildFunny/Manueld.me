# 版块 20 曲面积分计算

## B. 核心问题及解题方法总结

 **1. 简化一型曲面积分

方法 1: 奇偶对称简化

奇对称 $\left\{  \begin{array}{ll} \text{ 区域关于 }y\text{ oz }\text{ 面对称,} \\  \text{ 则 }{\iint }_{\sum }f\left( {x,y,z}\right) \mathrm{d}S = 0 \end{array}\right.$

偶对称 $\left\{  \begin{array}{ll} \text{ 区域关于 }y \geq  0\text{ 的对称,被积函数为 }x\text{的偶函数 } \\  \text{ 则 }{\iint }_{\sum }f\left( {x,y,z}\right) \mathrm{d}S = 2{\iint }_{{\sum }_{1}}f\left( {x,y,z}\right) \mathrm{d}S\text{,其中 }{\sum }_{1}\text{ 为 }\sum \text{ 的一半 } \end{array}\right.$

方法 2: 轮换对称简化

轮换对称 $\left\{  \begin{array}{ll} \text{ 积分区域关于 }x,y\text{ 轮换对称 } \\  {\iint }_{\sum }f\left( {x,y,z}\right) \mathrm{d}S = {\iint }_{\sum }f\left( {y,x,z}\right) \mathrm{d}S \end{array}\right.$

方法 3: 方程简化

方程简化: 可代入曲面方程化简被积函数

 **2. 计算第一型曲面积分

1. 运用二重积分计算: ${\iint }_{\sum }f\left( {x,y,z}\right) \mathrm{d}S = {\iint }_{{D}_{xy}}f\left\lbrack  {x,y,z\left( {x,y}\right) }\right\rbrack  \sqrt{1 + {z}_{x}^{\prime 2} + {z}_{y}^{\prime 2}}\mathrm{\;d}x\mathrm{\;d}y$

2. 应用。

① 计算曲面面积: $S = {\iint }_{\sum }\mathrm{d}S$

② 计算曲面质量: $m = {\iint }_{\sum }\mu \left( {x,y,z}\right) \mathrm{d}S = {\iint }_{{D}_{xy}}\mu \left\lbrack  {x,y,z\left( {x,y}\right) }\right\rbrack  \sqrt{1 + {z}_{x}^{\prime 2} + {z}_{y}^{\prime 2}}\mathrm{\;d}x\mathrm{\;d}y$

 **3. 简化与计算第二型曲面积分

1. 计算前先简化: 方程简化: 可代入曲面方程化简被积函数

2. 利用 $\rightarrow$ 重积分计算 $\left\{  \begin{array}{lll} {\iint }_{\sum }P\left( {x,y,z}\right) \mathrm{d}y\mathrm{\;d}z =  \pm  {\iint }_{{D}_{xy}}P\left\lbrack  {x\left( {y,z}\right) ,y,z}\right\rbrack  \mathrm{d}y\mathrm{\;d}z \\  {\iint }_{\sum }Q\left( {x,y,z}\right) \mathrm{d}x\mathrm{\;d}z =  \pm  {\iint }_{{D}_{xy}}Q\left\lbrack  {x,y\left( {x,z}\right) ,z}\right\rbrack  \mathrm{d}x\mathrm{\;d}z \\  {\iint }_{\sum }R\left( {x,y,z}\right) \mathrm{d}x\mathrm{\;d}y =  \pm  {\iint }_{{D}_{xy}}R\left\lbrack  {x,y,z\left( {x,y}\right) }\right\rbrack  \mathrm{d}x\mathrm{\;d}y \end{array}\right.$

 **4. 利用高斯公式计算曲面积分

1. 封闭外侧且单连通曲面的高斯公式:

封闭外侧: $${} \mathop{{\int\!\!\!\!\!\int}\mkern-21mu \bigcirc}\limits_\Sigma {Pdydz} + {Qdzdx} + {Rdxdy} = {\iiint }_{\Omega }\left( {\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}}\right) {dV}$$
2. 非封闭的处理方法

非封闭: $\left\{  \begin{array}{ll} \text{ 补面 } : {\iint }_{\sum }{Pdydz} + {Qdzdx} + {Rdxdy} = \left( {{\iint }_{\sum } + {\iint }_{{\sum }_{1}}}\right)  - {\iint }_{{\sum }_{1}} \\  \text{ 其中,运用高斯公式求 }{\iint }_{\sum } + {\iint }_{{\sum }_{1}}\text{,运用二重积分求 }{\int }_{\sum } \end{array}\right.$
