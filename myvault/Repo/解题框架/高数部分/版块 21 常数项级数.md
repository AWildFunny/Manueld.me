# 版块 21 常数项级数

## A. 实用解题工具

### 第 1 组: 常用参比级数

 3 组常用级数

常用级数 $\left\{  \begin{array}{lll} \text{ 等比级数 }\mathop{\sum }\limits_{{n = 1}}^{\infty }a{r}^{n - 1}\left\{  \begin{array}{ll} \text{ 收敛,}\left| r\right|  < 1 \\  \text{ 发散,}\left| r\right|  \geq  1 \end{array}\right. \\  p - \text{ 级数 }\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{1}{{n}^{p}}\left\{  \begin{array}{ll} \text{ 收敛,}p > 1\text{ 时 } \\  \text{ 发散,}p \leq  1\text{ 时 } \end{array}\right. \\  \text{ 交错版 }p - \text{ 级数 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{\left( -1\right) }^{n}\frac{1}{{n}^{p}}\left\{  \begin{array}{lll} p > 1,\text{ 绝对收敛 } \\  0 < p \leq  1,\text{ 条件收敛 } \\  p \leq  0,\text{ 交散 } \end{array}\right.  \end{array}\right.$

### 第 2 组: 级数的常用性质

1. 倍数不改变敛散性: 设 $c \neq  0$ 的常数,则 $\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n}$ 与 $\mathop{\sum }\limits_{{n = 1}}^{\infty }c{u}_{n}$ 有相同敛散性

2. 级数和的敛散性 $\left\{  \begin{array}{lll} \text{ 若 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n}\text{ 收敛,}\mathop{\sum }\limits_{{n = 1}}^{\infty }{v}_{n}\text{ 发散,则 }\mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {{u}_{n} \pm  {v}_{n}}\right) \text{ 发散. } \\  \text{ 若 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n},\mathop{\sum }\limits_{{n = 1}}^{\infty }{v}_{n}\text{ 均发散,则 }\mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {{u}_{n} \pm  {v}_{n}}\right) \text{ 敛散性不定. } \\  \text{ 若 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n} = s,\mathop{\sum }\limits_{{n = 1}}^{\infty }{v}_{n} = \sigma ,\text{ 则 }\mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {{u}_{n} \pm  {v}_{n}}\right)  = s \pm  \sigma . \end{array}\right.$

3. 加括号 $\left\{  \begin{array}{ll} \text{ 若 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n}\text{ 收敛,则 }\mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {{u}_{n} + {u}_{n + 1}}\right) \text{ 收敛 } \\  \text{ 若 }\mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {{u}_{n} + {u}_{n + 1}}\right) \text{ 收敛,则 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n}\text{ 不一定收敛 } \end{array}\right.$

### 第 3 组: 级数之和的敛散性

（1）绝对收敛 $\pm$ 条件收敛 $=$ 条件收敛;

（2）绝对收敛 $\pm$ 绝对收敛 $=$ 绝件收敛;

（3）条件收敛 $\pm$ 条件收敛 $=$ 条件收敛或绝对收敛;

例题 1: 设常数 $k > 0$ ,则级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }{\left( -1\right) }^{n}\frac{k + \sin n}{{n}^{2}}\left( \;\right)$ .

(A) 发散 (B) 绝对收敛

(C) 条件收敛 (D) 敛散性与 $k$ 有关

【解】选 B. 因为 ${\left( -1\right) }^{n}\frac{k + n}{{n}^{2}} = {\left( -1\right) }^{n}\frac{k}{{n}^{2}} + \frac{{\left( -1\right) }^{n}}{n}$ ,而 $\mathop{\sum }\limits_{{n = 1}}^{\infty }{\left( -1\right) }^{n}\frac{k}{{n}^{2}}$ 绝对收敛, $\mathop{\sum }\limits_{{n = 1}}^{\infty }{\left( -1\right) }^{n}\frac{\sin n}{{n}^{2}}$ 绝对收敛,所以 $\mathop{\sum }\limits_{{n = 1}}^{\infty }{\left( -1\right) }^{n}\frac{k + n}{{n}^{2}}$ 绝对收敛,选 B.

## B. 核心问题及解题方法总结

 **1. 利用正项级数审敛法判别正项级数敛散性

1. 比较审敛法: 设 $0 \leq  {u}_{n} \leq  {v}_{n}\left\{  \begin{array}{ll} \text{ 若 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{v}_{n}\text{ 收敛,则 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n}\text{ 收敛 } \\  \text{ 若 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n}\text{ 发散,则 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{v}_{n}\text{ 发散 } \end{array}\right.$

2. 比较审敛法极限形式: $0 \leq  {u}_{n},\mathop{\lim }\limits_{{n \rightarrow  \infty }}\frac{{u}_{n}}{{v}_{n}} = A\left\{  \begin{array}{ll} \text{ 若 }0 \leq  A <  + \infty ,\text{ 且 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{v}_{n}\text{ 收敛,则 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n}\text{ 收敛 } \\  \text{ 若 }0 < A \leq   + \infty ,\text{ 且 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{v}_{n}\text{ 发散,则 }\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n}\text{ 发散 } \end{array}\right.$

3. 比值,审敛法: 设 $0 \leq  {u}_{n}$ ,若 $\mathop{\lim }\limits_{{n \rightarrow  \infty }}\frac{{u}_{n + 1}}{{u}_{n}} = \rho \left\{  \begin{array}{lll} \rho  > 1\text{时,}\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n}\text{ 发散 } \\  \rho  = 1\text{ 时,}\text{ 方法失效 } \\  \rho  < 1\text{ 时,}\mathop{\sum }\limits_{{n = 1}}^{\infty }{u}_{n}\text{ 收敛 } \end{array}\right.$

 **2. 利用莱布尼茨法判别交错级数敛散性

莱布尼茨审敛法:

若 $\mathop{\sum }\limits_{{n = 1}}^{\infty }{\left( -1\right) }^{n - 1}{u}_{n},\left( {{u}_{n} > 0}\right)$ 满足条件

(1) ${u}_{n} \geq  {u}_{n + 1},\left( {n = 1,2,\cdots \cdots }\right)  : \left( 2\right) \mathop{\lim }\limits_{{n \rightarrow  \infty }}{u}_{n} = 0$

则交错级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }{\left( -1\right) }^{n - 1}{u}_{n}$ 收敛
