# 版块 22 幂级数
## A. 实用解题工具

### 第 1 组: 级数求和基本公式

1. 最重要公式: $\mathop{\sum }\limits_{{n = 0}}^{\infty }{x}^{n} = 1 + x + {x}^{2} + \cdots  + {x}^{n} + \cdots  = \frac{1}{1 - x}$

2. 次重要公式: $\mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{n + 1}}{n + 1} = x - \frac{{x}^{2}}{2} + \frac{{x}^{3}}{3} - \cdots  + {\left( -1\right) }^{n}\frac{{x}^{n + 1}}{n + 1} + \cdots  = \ln \left( {1 + x}\right)$

3. 带阶乘公式: $\left\{  \begin{array}{llll} \mathop{\sum }\limits_{{n = 0}}^{\infty }\frac{{x}^{n}}{n!} = 1 + x + \frac{{x}^{2}}{2!} + \cdots  + \frac{{x}^{n}}{n!} + \cdots  = {e}^{x} \\  \mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{{2n} + 1}}{\left( {{2n} + 1}\right) !} = x - \frac{{x}^{3}}{3!} + \cdots  + {\left( -1\right) }^{n}\frac{{x}^{{2n} + 1}}{\left( {{2n} + 1}\right) !} + \cdots  = \sin x \\  \mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{2n}}{\left( {2n}\right) !} = 1 - \frac{{x}^{2}}{2!} + \frac{{x}^{4}}{4!} - \cdots  + {\left( -1\right) }^{n}\frac{{x}^{2n}}{\left( {2n}\right) !} + \cdots  = \cos x \\  1 + \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{a\left( {a - 1}\right) \cdots \left( {a - n + 1}\right) }{n!}{x}^{n} = 1 + {ax} + \cdots  + \frac{a\left( {a - 1}\right) \cdots \left( {a - n + 1}\right) }{n!}{x}^{n}\cdots  = {\left( 1 + x\right) }^{a} \end{array}\right.$

## B. 核心问题及解题方法总结

 **1. 求解收敛半径

阿贝尔定理 $\left\{  \begin{array}{ll} \text{ 幂级数 }\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}\text{ 在 }{x}_{0} \neq  0\text{ 处收敛,则其在满足 }\left| x\right|  < \left| {x}_{0}\right| \text{ 的所有点处绝对收敛 } \\  \text{ 幂级数 }\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}\text{ 在 }{x}_{1} \neq  0\text{ 处发散,则其在满足 }\left| x\right|  > \left| {x}_{1}\right| \text{ 的所有点处发散 } \end{array}\right.$

 收敛半径求法
  $\left\{  \begin{array}{lll} \text{ 幂级数 }\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}\text{ 收敛半径: }\mathop{\lim }\limits_{{n \rightarrow  \infty }}\left| \frac{{a}_{n + 1}}{{a}_{n}}\right|  = \rho  \Rightarrow  R = \frac{1}{\rho }. \\  \text{ 幂级数 }\mathop{\sum }\limits_{{n = 0}}^{\infty }{u}_{n}\left( x\right) \text{ 的收敛区间: }\mathop{\lim }\limits_{{n \rightarrow  \infty }}\left| \frac{{u}_{n + 1}}{{u}_{n}}\right|  = \rho \left( x\right)  < 1\text{ 或 }\mathop{\lim }\limits_{{n \rightarrow  \infty }}\sqrt[n]{\left| {u}_{n}\left( x\right) \right| } = \rho \left( x\right)  < 1,\text{ 解出的 }x\text{ 范围 } \\  \text{ 幂级数 }\mathop{\sum }\limits_{{n = 0}}^{\infty }{u}_{n}\left( x\right) \text{ 的收敛域: 在收敛区间的基础上,再额外判断区间端点的敛散性 } \end{array}\right.$

例题 1. 幂级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {\frac{1}{{n}^{2}} + \frac{1}{{2}^{n}}}\right) {x}^{n}$ 的收敛域为___.

【解】首先,求 $\rho$

$$
\rho  = \mathop{\lim }\limits_{{n \rightarrow  \infty }}\left| \frac{\frac{1}{{\left( n + 1\right) }^{2}} + \frac{1}{{2}^{n + 1}}}{\frac{1}{{n}^{2}} + \frac{1}{{2}^{n}}}\right|  = \mathop{\lim }\limits_{{n \rightarrow  \infty }}\frac{1 + \frac{{\left( n + 1\right) }^{2}}{{2}^{n + 1}}}{1 + \frac{{n}^{2}}{{2}^{n}}}\frac{1}{{\left( 1 + \frac{1}{n}\right) }^{2}} = 1.
$$

其次,写出幂级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {\frac{1}{{n}^{2}} + \frac{1}{{2}^{n}}}\right) {x}^{n}$ 的收敛半径与收敛区间.

其收敛半径 $R = \frac{1}{\rho } = 1$ ,收敛区间为 $\left( {-1,1}\right)$ .

最后, 确定收敛区间端点的敛散性, 写出收敛域.

当 $x =  - 1$ 时, $\mathop{\sum }\limits_{{n = 1}}^{\infty }{\left( -1\right) }^{n}\left( {\frac{1}{{n}^{2}} + \frac{1}{{2}^{n}}}\right)$ 收敛; 当 $x = 1,\mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {\frac{1}{{n}^{2}} + \frac{1}{{2}^{n}}}\right)$ 收敛. 故该幂级数的收敛域为 $\left\lbrack  {-1,1}\right\rbrack$ .

 **2. 收敛半径的变化

1. 若幂级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }{a}_{n}{x}^{n}$ 的收敛半径为 $R$ ,则:

（1） ${a}_{n}$ 前乘以或除以 $n$ 的多项式,收敛半径不改变。示例: $\mathop{\sum }\limits_{{n = 1}}^{\infty }n{a}_{n}{x}^{n},\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{1}{n}{a}_{n}{x}^{n},\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{n}^{3} - 2}{{n}^{2} + n}{a}_{n}{x}^{n}$ 半径依然为  $R$ 。

（2） ${a}_{n}$ 前乘以指数 ${k}^{n}$ ,收敛半径变为 $\frac{R}{k}$ 。示例: $\mathop{\sum }\limits_{{n = 1}}^{\infty }{a}_{n}{2}^{n}{x}^{n},\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{1}{{2}^{n}}{a}_{n}{x}^{n}$ 的收敛半径分别为 $\frac{R}{2},{2R}$

 **3. 条件收敛确定半径

条件收敛确定收敛半径: 若幂级数 $\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}$ 在点 $x = c$ 处条件收敛,则级数的收敛半径必为 $c$

因为如果 $R < c$ ,则幂级数 $\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}$ 在点 $x = c$ 处应发散,矛盾; 如果 $R > c$ ,则幂级数 $\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}$ 在 $x = c$ 处应绝对收敛,矛盾。所以收敛半径必为 $c$

 **4.级数展开的方法

1. 直接法: $f\left( x\right)  = f\left( {x}_{0}\right)  + {f}^{\prime }\left( {x}_{0}\right) \left( {x - {x}_{0}}\right)  + \frac{{f}^{\prime \prime }\left( {x}_{0}\right) }{2!}{\left( x - {x}_{0}\right) }^{2} + \cdots$

2. 间接法: $\left\{  \begin{array}{llllll} \frac{1}{1 - x} = 1 + x + {x}^{2} + \cdots  + {x}^{n} + \cdots  = \mathop{\sum }\limits_{{n = 0}}^{\infty }{x}^{n},\left( {-1,1}\right) \\  {e}^{x} = 1 + x + \frac{{x}^{2}}{2!} + \cdots  + \frac{{x}^{n}}{n!} + \cdots  = \mathop{\sum }\limits_{{n = 0}}^{\infty }\frac{{x}^{n}}{n!},\left( {-\infty , + \infty }\right) \\  \sin x = x - \frac{{x}^{3}}{3!} + \cdots  + {\left( -1\right) }^{n}\frac{{x}^{{2n} + 1}}{\left( {{2n} + 1}\right) !} + \cdots  = \mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{{2n} + 1}}{\left( {{2n} + 1}\right) !},\left( {-\infty , + \infty }\right) \\  \cos x = 1 - \frac{{x}^{2}}{2!} + \frac{{x}^{4}}{4!} - \cdots  + {\left( -1\right) }^{n}\frac{{x}^{2n}}{\left( {2n}\right) !} + \cdots  = \mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{2n}}{\left( {2n}\right) !},\left( {-\infty , + \infty }\right) \\  \ln \left( {1 + x}\right)  = x - \frac{{x}^{2}}{2} + \frac{{x}^{3}}{3} - \cdots  + {\left( -1\right) }^{n}\frac{{x}^{n + 1}}{n + 1} + \cdots \mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{n + 1}}{n + 1},\left( {-1,1}\right) \\  {\left( 1 + x\right) }^{a} = 1 + {ax} + \frac{a\left( {a - 1}\right) }{2!}{x}^{2} + \cdots  + \frac{a\left( {a - 1}\right) \cdots \left( {a - n + 1}\right) }{n!}{x}^{n}\cdots ,\left( {-1,1}\right)  \end{array}\right.$

 **5. 求解 $n$ 在分子的级数

1. $n$ 在分子的求解思路: 先积分再求导 (如 $\mathop{\sum }\limits_{{n = 1}}^{\infty }n{x}^{n}$ 变形为 $x\mathop{\sum }\limits_{{n = 1}}^{\infty }n{x}^{n - 1}$ ,先积分再求导)

2. 积分时幂次比系数小 1,即: $n{x}^{n - 1}$ 的形式。如果是 $n{x}^{n}$ 形式,则需要将多余的 $x$ 提至求和符号外。如果是  $n{x}^{n - 2}$ 形式,则需要将 $\frac{1}{x}$ 提至求和符号外。
