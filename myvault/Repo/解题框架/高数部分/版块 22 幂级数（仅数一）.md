# 版块 22 幂级数（仅数一）
## A. 实用解题工具

### 第 1 组: 级数求和基本公式

1. 最重要公式: $\mathop{\sum }\limits_{{n = 0}}^{\infty }{x}^{n} = 1 + x + {x}^{2} + \cdots  + {x}^{n} + \cdots  = \frac{1}{1 - x}$

2. 次重要公式: $\mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{n + 1}}{n + 1} = x - \frac{{x}^{2}}{2} + \frac{{x}^{3}}{3} - \cdots  + {\left( -1\right) }^{n}\frac{{x}^{n + 1}}{n + 1} + \cdots  = \ln \left( {1 + x}\right)$

3. 带阶乘公式: $\left\{  \begin{array}{llll} \mathop{\sum }\limits_{{n = 0}}^{\infty }\frac{{x}^{n}}{n!} = 1 + x + \frac{{x}^{2}}{2!} + \cdots  + \frac{{x}^{n}}{n!} + \cdots  = {e}^{x} \\  \mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{{2n} + 1}}{\left( {{2n} + 1}\right) !} = x - \frac{{x}^{3}}{3!} + \cdots  + {\left( -1\right) }^{n}\frac{{x}^{{2n} + 1}}{\left( {{2n} + 1}\right) !} + \cdots  = \sin x \\  \mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{2n}}{\left( {2n}\right) !} = 1 - \frac{{x}^{2}}{2!} + \frac{{x}^{4}}{4!} - \cdots  + {\left( -1\right) }^{n}\frac{{x}^{2n}}{\left( {2n}\right) !} + \cdots  = \cos x \\  1 + \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{a\left( {a - 1}\right) \cdots \left( {a - n + 1}\right) }{n!}{x}^{n} = 1 + {ax} + \cdots  + \frac{a\left( {a - 1}\right) \cdots \left( {a - n + 1}\right) }{n!}{x}^{n}\cdots  = {\left( 1 + x\right) }^{a} \end{array}\right.$

## B. 核心问题及解题方法总结

 **1. 求解收敛半径

阿贝尔定理 $\left\{  \begin{array}{ll} \text{ 幂级数 }\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}\text{ 在 }{x}_{0} \neq  0\text{ 处收敛,则其在满足 }\left| x\right|  < \left| {x}_{0}\right| \text{ 的所有点处绝对收敛 } \\  \text{ 幂级数 }\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}\text{ 在 }{x}_{1} \neq  0\text{ 处发散,则其在满足 }\left| x\right|  > \left| {x}_{1}\right| \text{ 的所有点处发散 } \end{array}\right.$

 收敛半径求法
  $\left\{  \begin{array}{lll} \text{ 幂级数 }\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}\text{ 收敛半径: }\mathop{\lim }\limits_{{n \rightarrow  \infty }}\left| \frac{{a}_{n + 1}}{{a}_{n}}\right|  = \rho  \Rightarrow  R = \frac{1}{\rho }. \\  \text{ 幂级数 }\mathop{\sum }\limits_{{n = 0}}^{\infty }{u}_{n}\left( x\right) \text{ 的收敛区间: }\mathop{\lim }\limits_{{n \rightarrow  \infty }}\left| \frac{{u}_{n + 1}}{{u}_{n}}\right|  = \rho \left( x\right)  < 1\text{ 或 }\mathop{\lim }\limits_{{n \rightarrow  \infty }}\sqrt[n]{\left| {u}_{n}\left( x\right) \right| } = \rho \left( x\right)  < 1,\text{ 解出的 }x\text{ 范围 } \\  \text{ 幂级数 }\mathop{\sum }\limits_{{n = 0}}^{\infty }{u}_{n}\left( x\right) \text{ 的收敛域: 在收敛区间的基础上,再额外判断区间端点的敛散性 } \end{array}\right.$

例题 1. 幂级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {\frac{1}{{n}^{2}} + \frac{1}{{2}^{n}}}\right) {x}^{n}$ 的收敛域为___.

【解】首先,求 $\rho$

$$
\rho  = \mathop{\lim }\limits_{{n \rightarrow  \infty }}\left| \frac{\frac{1}{{\left( n + 1\right) }^{2}} + \frac{1}{{2}^{n + 1}}}{\frac{1}{{n}^{2}} + \frac{1}{{2}^{n}}}\right|  = \mathop{\lim }\limits_{{n \rightarrow  \infty }}\frac{1 + \frac{{\left( n + 1\right) }^{2}}{{2}^{n + 1}}}{1 + \frac{{n}^{2}}{{2}^{n}}}\frac{1}{{\left( 1 + \frac{1}{n}\right) }^{2}} = 1.
$$

其次,写出幂级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {\frac{1}{{n}^{2}} + \frac{1}{{2}^{n}}}\right) {x}^{n}$ 的收敛半径与收敛区间.

其收敛半径 $R = \frac{1}{\rho } = 1$ ,收敛区间为 $\left( {-1,1}\right)$ .

最后, 确定收敛区间端点的敛散性, 写出收敛域.

当 $x =  - 1$ 时, $\mathop{\sum }\limits_{{n = 1}}^{\infty }{\left( -1\right) }^{n}\left( {\frac{1}{{n}^{2}} + \frac{1}{{2}^{n}}}\right)$ 收敛; 当 $x = 1,\mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {\frac{1}{{n}^{2}} + \frac{1}{{2}^{n}}}\right)$ 收敛. 故该幂级数的收敛域为 $\left\lbrack  {-1,1}\right\rbrack$ .

 **2. 收敛半径的变化

1. 若幂级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }{a}_{n}{x}^{n}$ 的收敛半径为 $R$ ,则:

（1） ${a}_{n}$ 前乘以或除以 $n$ 的多项式,收敛半径不改变。示例: $\mathop{\sum }\limits_{{n = 1}}^{\infty }n{a}_{n}{x}^{n},\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{1}{n}{a}_{n}{x}^{n},\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{n}^{3} - 2}{{n}^{2} + n}{a}_{n}{x}^{n}$ 半径依然为  $R$ 。

（2） ${a}_{n}$ 前乘以指数 ${k}^{n}$ ,收敛半径变为 $\frac{R}{k}$ 。示例: $\mathop{\sum }\limits_{{n = 1}}^{\infty }{a}_{n}{2}^{n}{x}^{n},\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{1}{{2}^{n}}{a}_{n}{x}^{n}$ 的收敛半径分别为 $\frac{R}{2},{2R}$

2. 变化后的半径按照 1 中规律, 但变化后区间的端点需要重新验证是否收敛

3. 幂级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }{x}^{n}$ 的收敛半径 $R = 1$

 **3. 条件收敛确定半径

条件收敛确定收敛半径: 若幂级数 $\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}$ 在点 $x = c$ 处条件收敛,则级数的收敛半径必为 $c$

因为如果 $R < c$ ,则幂级数 $\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}$ 在点 $x = c$ 处应发散,矛盾; 如果 $R > c$ ,则幂级数 $\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}$ 在 $x = c$ 处应绝对收敛,矛盾。所以收敛半径必为 $c$

1. 设级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }{a}_{n}$ 条件收敛,则幂级数 $\mathop{\sum }\limits_{{n = 0}}^{\infty }\frac{{a}_{n}}{n + 1}{\left( x - 2\right) }^{n}$ 的收敛区间为___.

【解】级数 $\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}$ 条件收敛,幂级数 $\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}$ 在点 $x = 1$ 处条件收敛,故它的收敛半径 $R = 1$ .

由于逐项积分所得幂级数与原幂级数有相同的收敛半径,故幂级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{a}_{n}}{n + 1}{\left( x - 2\right) }^{n}$ 与幂级数  $\mathop{\sum }\limits_{{n = 1}}^{\infty }{a}_{n}{\left( x - 2\right) }^{n + 1}$ 的收敛半径均为 1,所以所求收敛区间为 $\left( {2 - 1,2 + 1}\right)  = \left( {1,3}\right)$

 **4.级数展开的方法

1. 直接法: $f\left( x\right)  = f\left( {x}_{0}\right)  + {f}^{\prime }\left( {x}_{0}\right) \left( {x - {x}_{0}}\right)  + \frac{{f}^{\prime \prime }\left( {x}_{0}\right) }{2!}{\left( x - {x}_{0}\right) }^{2} + \cdots$

2. 间接法: $\left\{  \begin{array}{llllll} \frac{1}{1 - x} = 1 + x + {x}^{2} + \cdots  + {x}^{n} + \cdots  = \mathop{\sum }\limits_{{n = 0}}^{\infty }{x}^{n},\left( {-1,1}\right) \\  {e}^{x} = 1 + x + \frac{{x}^{2}}{2!} + \cdots  + \frac{{x}^{n}}{n!} + \cdots  = \mathop{\sum }\limits_{{n = 0}}^{\infty }\frac{{x}^{n}}{n!},\left( {-\infty , + \infty }\right) \\  \sin x = x - \frac{{x}^{3}}{3!} + \cdots  + {\left( -1\right) }^{n}\frac{{x}^{{2n} + 1}}{\left( {{2n} + 1}\right) !} + \cdots  = \mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{{2n} + 1}}{\left( {{2n} + 1}\right) !},\left( {-\infty , + \infty }\right) \\  \cos x = 1 - \frac{{x}^{2}}{2!} + \frac{{x}^{4}}{4!} - \cdots  + {\left( -1\right) }^{n}\frac{{x}^{2n}}{\left( {2n}\right) !} + \cdots  = \mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{2n}}{\left( {2n}\right) !},\left( {-\infty , + \infty }\right) \\  \ln \left( {1 + x}\right)  = x - \frac{{x}^{2}}{2} + \frac{{x}^{3}}{3} - \cdots  + {\left( -1\right) }^{n}\frac{{x}^{n + 1}}{n + 1} + \cdots \mathop{\sum }\limits_{{n = 0}}^{\infty }{\left( -1\right) }^{n}\frac{{x}^{n + 1}}{n + 1},\left( {-1,1}\right) \\  {\left( 1 + x\right) }^{a} = 1 + {ax} + \frac{a\left( {a - 1}\right) }{2!}{x}^{2} + \cdots  + \frac{a\left( {a - 1}\right) \cdots \left( {a - n + 1}\right) }{n!}{x}^{n}\cdots ,\left( {-1,1}\right)  \end{array}\right.$

 **5. 求解 $n$ 在分子的级数

1. $n$ 在分子的求解思路: 先积分再求导 (如 $\mathop{\sum }\limits_{{n = 1}}^{\infty }n{x}^{n}$ 变形为 $x\mathop{\sum }\limits_{{n = 1}}^{\infty }n{x}^{n - 1}$ ,先积分再求导)

2. 积分时幂次比系数小 1,即: $n{x}^{n - 1}$ 的形式。如果是 $n{x}^{n}$ 形式,则需要将多余的 $x$ 提至求和符号外。如果是  $n{x}^{n - 2}$ 形式,则需要将 $\frac{1}{x}$ 提至求和符号外。

 **6. 求解 $n$ 在分母的级数

1. $n$ 在分母的求解思路: 先求导再积分 (如 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{x}^{n}}{n}$ ,先求导再积分)

2. 求导时幂次与系数相乘为 1,即: $\frac{1}{n}{x}^{n}$ 的形式。如果是 $\frac{1}{n}{x}^{n + 1}$ 形式,则需要将多余的 $x$ 提至求和符号外。如果是 $\frac{1}{n}{x}^{n - 1}$ 形式,则需要将 $\frac{1}{x}$ 提至求和符号外。

例题 1. 求幂级数 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{\left( -1\right) }^{n - 1}{x}^{{2n} + 1}}{n\left( {{2n} - 1}\right) }$ 的收敛域及和函数 $s\left( x\right)$ .

【解】记 ${u}_{n}\left( x\right)  = \frac{{\left( -1\right) }^{n - 1}{x}^{{2n} + 1}}{n\left( {{2n} - 1}\right) }$ ,则 $\mathop{\lim }\limits_{{n \rightarrow  \infty }}\left| \frac{{u}_{n + 1}\left( x\right) }{{u}_{n}\left( x\right) }\right|  = \mathop{\lim }\limits_{{n \rightarrow  \infty }}\frac{\left| \frac{{\left( -1\right) }^{n}{x}^{{2n} + 3}}{\left( {n + 1}\right) \left( {{2n} + 1}\right) }\right| }{\frac{{\left( -1\right) }^{n - 1}{x}^{{2n} + 1}}{n\left( {{2n} - 1}\right) }} = {\left| x\right| }^{2}$ .

所以当 ${\left| x\right| }^{2} < 1$ ,即 $\left| x\right|  < 1$ 时,所给幂级数收敛; 当 $\left| x\right|  > 1$ 时,所给幂级数发散;

当 $x =  \pm  1$ 时,所给幂级数为 $\frac{{\left( -1\right) }^{n - 1}}{n\left( {{2n} - 1}\right) },\frac{{\left( -1\right) }^{n}}{n\left( {{2n} - 1}\right) }$ ,均收敛,

故所给幂级数的收敛域为 $\left\lbrack  {-1,1}\right\rbrack$

在 $\left( {-1,1}\right)$ 内, $s\left( x\right)  = \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{\left( -1\right) }^{n - 1}{x}^{{2n} + 1}}{n\left( {{2n} - 1}\right) } = {2x}\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{\left( -1\right) }^{n - 1}{x}^{2n}}{\left( {{2n} - 1}\right) \left( {2n}\right) } = {2x}{s}_{1}\left( x\right)$ ,

而 ${s}_{1}^{\prime }\left( x\right)  = \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{\left( -1\right) }^{n - 1}{x}^{{2n} - 1}}{{2n} - 1},{s}_{1}^{\prime \prime }\left( x\right)  = \mathop{\sum }\limits_{{n = 1}}^{\infty }{\left( -1\right) }^{n - 1}{x}^{{2n} - 2} = \frac{1}{1 + {x}^{2}}$ ,

所以 ${s}_{1}^{\prime }\left( x\right)  - {s}_{1}^{\prime }\left( 0\right)  = {\int }_{0}^{x}{s}_{1}^{\prime \prime }\left( t\right) \mathrm{d}t = {\int }_{0}^{x}\frac{1}{1 + {t}^{2}}\mathrm{\;d}t = \arctan x$ ,又 ${s}_{1}^{\prime }\left( 0\right)  = 0$ ,

于是 ${s}_{1}^{\prime }\left( x\right)  = \arctan x$ . 同理

${s}_{1}\left( x\right)  - {s}_{1}\left( 0\right)  = {\int }_{0}^{x}{s}_{1}^{\prime }\left( t\right) \mathrm{d}t = {\int }_{0}^{x}\arctan t\mathrm{\;d}t$

$= {\left. t\arctan t\right| }_{0}^{x} - {\int }_{0}^{x}\frac{t}{1 + {t}^{2}}\mathrm{\;d}t = x\arctan x - \frac{1}{2}\ln \left( {1 + {x}^{2}}\right) ,$

又 ${s}_{1}\left( 0\right)  = 0$ ,所以 ${s}_{1}\left( x\right)  = x\arctan x - \frac{1}{2}\ln \left( {1 + {x}^{2}}\right)$ .

故 $s\left( x\right)  = 2{x}^{2}\arctan x - x\ln \left( {1 + {x}^{2}}\right) ,x \in  \left( {-1,1}\right)$ .

由于所给幂级数在 $x =  \pm  1$ 处都收敛,且 $s\left( x\right)  = 2{x}^{2}\arctan x - x\ln \left( {1 + {x}^{2}}\right)$ 在 $x =  \pm  1$ 处都

连续,所以 $s\left( x\right)$ 在 $x =  \pm  1$ 成立,即 $s\left( x\right)  = 2{x}^{2}\arctan x - x\ln \left( {1 + {x}^{2}}\right) ,x \in  \left\lbrack  {-1,1}\right\rbrack$ .

 **7. 利用幂级数求解常数项级数之和

方法: 带 ${a}^{n}$ 或 ${1}^{n}$ 的常数项级数之和,可化为幂级数求解。

例题 $1 \cdot  \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{\left( {n + 1}\right) {2}^{n}}{n!} =$

【解】设 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{n + 1}{n!}{x}^{n} = \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{n}{n!}{x}^{n} + \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{x}^{n}}{n!}$

又因为 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{n}{n!}{x}^{n} = \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{1}{\left( {n - 1}\right) !}{x}^{n} = x\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{1}{\left( {n - 1}\right) !}{x}^{n - 1} = x\mathop{\sum }\limits_{{n = 0}}^{\infty }\frac{1}{n!}{x}^{n} = x{\mathrm{e}}^{x}$ ,

且 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{{x}^{n}}{n!} =  - 1 + \mathop{\sum }\limits_{{n = 0}}^{\infty }\frac{{x}^{n}}{n!} =  - 1 + {\mathrm{e}}^{x}$ 所以 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{n + 1}{n!}{x}^{n} = \left( {x + 1}\right) {\mathrm{e}}^{x} - 1$

所以 $\mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{\left( {n + 1}\right) {2}^{n}}{n!} = \mathop{\sum }\limits_{{n = 1}}^{\infty }\frac{n + 1}{n!}{2}^{n} = \left( {1 + 2}\right) {\mathrm{e}}^{2} - 1 = 3{\mathrm{e}}^{2} - 1$ .

 **8. 求解含递推关系的级数之和

方法: 出现递推关系, 首选构建微分方程求级数之和

例题 1. 设数列 $\left\{  {a}_{n}\right\}$ 满足条件: ${a}_{0} = 3,{a}_{1} = 1,{a}_{n - 2} - n\left( {n - 1}\right) {a}_{n} = 0\left( {n \geq  2}\right) ,S\left( x\right)$ 是幂级数 $\mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n}$ 的和函数。
（1）证明: ${S}^{\prime \prime }\left( x\right)  - S\left( x\right)  = 0$ ; (2) 求 $S\left( x\right)$ 的表达式.

【解】（1）设 $S\left( x\right)  = \mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n},{S}^{\prime }\left( x\right)  = \mathop{\sum }\limits_{{n = 1}}^{\infty }{a}_{n}n{x}^{n - 1},{S}^{\prime }\left( x\right)  = \mathop{\sum }\limits_{{n = 2}}^{\infty }{a}_{n}n\left( {n - 1}\right) {x}^{n - 2}$ ,

因为 ${a}_{n - 2} - n\left( {n - 1}\right) {a}_{n} = 0$ ,因此

${S}^{\prime \prime }\left( x\right)  = \mathop{\sum }\limits_{{n = 2}}^{\infty }{a}_{n}n\left( {n - 1}\right) {x}^{n - 2} = \mathop{\sum }\limits_{{n = 2}}^{\infty }{a}_{n - 2}{x}^{n - 2} = \mathop{\sum }\limits_{{n = 0}}^{\infty }{a}_{n}{x}^{n} = S\left( x\right) ;$

（2）方程 ${S}^{\prime \prime }\left( x\right)  - S\left( x\right)  = 0$ 的特征方程为 ${\lambda }^{2} - 1 = 0$ ,

解得 ${\lambda }_{1} =  - 1,{\lambda }_{1} = 1$ ,所以 $S\left( x\right)  = {c}_{1}{e}^{-x} + {c}_{2}{e}^{x}$ ,

又 ${a}_{0} = S\left( 0\right)  = 3 \Rightarrow  {c}_{1} + {c}_{2} = 3,{a}_{1} = {s}^{\prime }\left( 0\right)  = {c}_{2} - {c}_{1} = 1$ ,

解得 ${c}_{1} = 1,{c}_{2} = 2$ ,所以 $s\left( x\right)  = {e}^{-x} + 2{e}^{x}$ 。

 **9. 狄利克雷收敛定理 (仅数学一)

设周期为 ${2\pi }$ 的函数 $f\left( x\right)$ 在 $\left\lbrack  {-\pi ,\pi }\right\rbrack$ 上满足条件:

(1)除有限个第一类间断点外都连续;(2)只有有限个单调区间,

$f\left( x\right)$ 的以 ${2\pi }$ 为周期的傅立叶级数在 $\left\lbrack  {-\pi ,\pi }\right\rbrack$ 上收敛,

且 $\frac{{a}_{0}}{2} + \mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {{a}_{n}\cos {nx} + {b}_{n}\sin {nx}}\right)  = \frac{1}{2}\left\lbrack  {f\left( {x}^{ - }\right)  + f\left( {x}^{ + }\right) }\right\rbrack$

【笔记】该结论不局限于周期为 ${2\pi }$ 的函数 $f\left( x\right)$ ,其他周期的周期函数也有类似的结论。

 **10.展开为傅里叶级数的方法 (仅数学一)

1. $\left\lbrack  {-l,l}\right\rbrack$ 上展开为傅里叶级数 $\left\{  \begin{array}{lll} {a}_{n} = \frac{1}{l}{\int }_{-l}^{l}f\left( x\right) \cos \frac{n\pi }{l}{xdx},\left( {n = 0,1,2\cdots }\right) \\  {b}_{n} = \frac{1}{l}{\int }_{-l}^{l}f\left( x\right) \sin \frac{n\pi }{l}{xdx},\left( {n = 0,1,2\cdots }\right) \\  f\left( x\right)  \sim  \frac{1}{2}{a}_{0} + \mathop{\sum }\limits_{{n = 1}}^{\infty }\left( {{a}_{n}\cos \frac{n\pi }{l}x + {b}_{n}\sin \frac{n\pi }{l}x}\right)  \end{array}\right.$
